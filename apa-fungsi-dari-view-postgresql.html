Bahasa kueri terstruktur atau SQL adalah bahasa kueri yang dirancang untuk membantu Anda mengambil dan mengelola informasi dalam database Anda. Bagi mereka yang baru mengenal dunia TI, biasanya didefinisikan sebagai bahasa yang digunakan untuk mengakses  data secara relasional. SQL atau bahasa query standar adalah bahasa pemrograman yang digunakan untuk mengambil, memodifikasi, dan memanipulasi data  relasional. Tentu saja, bahasa komputer dari database relasional ini didasarkan pada standar yang diterbitkan oleh American National Standards Institute (ANSI). SQL adalah bahasa query, sedangkan MySQL adalah database relasional yang menggunakan SQL untuk query database. Anda dapat menggunakan SQL untuk mengakses, memperbarui, dan memanipulasi data yang disimpan dalam database.
