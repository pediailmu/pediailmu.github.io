22. Tuliskan Langkah-langkah mengubah format dokumen Ms. Word menjadi Pdf dengan aplikasi google dokumen!T_T ​


Jawaban:

Berikut adalah langkah-langkah untuk mengubah format dokumen Microsoft Word menjadi PDF dengan menggunakan aplikasi Google Dokumen:

1. Buka browser dan kunjungi situs Google Dokumen di https://docs.google.com.

2. Jika Anda belum login ke akun Google Anda, masuklah terlebih dahulu dengan menggunakan akun Google yang Anda miliki.

3. Setelah masuk ke Google Dokumen, klik tombol "Beranda" di kanan atas halaman.

4. Pilih "Upload" untuk mengunggah dokumen Word yang ingin Anda ubah menjadi PDF.

5. Pilih file dokumen Word yang ingin Anda ubah dan klik "Buka" untuk memulai proses pengunggahan.

6. Setelah dokumen berhasil diunggah, klik tombol "File" di bagian atas halaman.

7. Di menu drop-down, pilih "Unduh" lalu pilih "PDF Dokumen" untuk mengonversi dokumen menjadi format PDF.

8. Setelah memilih opsi PDF Dokumen, Google Dokumen akan memulai proses konversi dokumen Word menjadi PDF.

9. Setelah proses konversi selesai, dokumen PDF akan otomatis diunduh ke komputer Anda.

10. Buka file PDF yang diunduh untuk memastikan bahwa konversi selesai dengan benar.

Dengan mengikuti langkah-langkah di atas, Anda dapat mengubah format dokumen Microsoft Word menjadi PDF dengan menggunakan aplikasi Google Dokumen.

31. DOKUMENSANGAT RAHASIAMATEΜΑΤΙΚΑ1. 8 + (-10) -3x2​


Penjelasan dengan langkah-langkah:

1.8+(-10)=8-(-3×2)=2

8 plus minus 10 = 8 dikurangi 3 dikali 2 minus sama dengan 2

Jawab:

8 + (-10) - 3 x 2 = - 8

Penjelasan dengan langkah-langkah:

Menghitung operasional bilangan

Mapel : Matematika

Hasil dari 8 + (-10) - 3 x 2 adalah .....

Pertama yang dihitung adalah perkaliannya terlebih dahulu

8 + (-10) - 3 x 2

= 8 + (-10) - 6

= 8 - 10 - 6

= -2 - 6

= - 8

Jadi, hasil dari 8 + (-10) - 3 x 2 adalah - 8

36. cara agar bisa merubah file dokumen menjadi file ebook dengan format pdf ?


Cara agar bisa merubah file dokumen menjadi file ebook dengan format pdf adalah :

Buka file berekstensi dokumen yang akan Anda ubah menjadi file berekstensi PDF (dalam kasus ini penulis menggunakan aplikasi Ms. Word 2013)Buka tab file, dan pilih eksporPilih "Buat PDF/XPS"Lalu beri nama file berekstensi PDF yang Anda inginkanDan klik "terbitkan"Selamat file berekstensi dokumen Anda sudah menjadi file berformat PDF

PEMBAHASAN :

Sebelum menjawab soal tersebut, mari kita mengenal terlebih dahulu file yang berekstensi pdf ^_^

→ Portable Document Format atau biasa disingkat dengan PDF, merupakan suatu ekstensi yang dikembangkan oleh perusahaan Adobe Inc pada tahun 1993.

→ Ektensi PDF ini memungkinkan user untuk dapat merubah baik file berupa dokumen (doc atau docx) dan gambar menjadi satu file berekstensi PDF yang praktis dengan kompabilitas tinggi.

→ Untuk dapat membuka file PDF, diperlukan aplikasi reader khusus seperti Adobe reader, Foxit PDF, Nitro PDF, dan banyak aplikasi pihak ke 3 lainnya.

→ Saat ini tingkat penggunaan file berekstensi PDF sangat tinggi, disebabkan oleh beberapa kelebihan yang dimilikinya, seperti :

membutuhkan ruang penyimpanan (memori) yang relatif kecil dibanding ekstensi lainnyafile dokumen berupa tulisan atau grafik akan dianggap sebagai gambar, sehingga memudahkan untuk perpindahan halaman saat membaca ebooktidak adanya kekhawatiran terhadap kemungkinan salah tekan atau perubahan konten ebook yang tidak disengaja, karena file pdf akan menganggap isinya sebagai sebuah gambar yang solid, dan hanya bisa di edit saat memiliki aplikasi pdf editor yang rata-rata berbayarlebih tahan akan kemungkinan kerusakan dokumen, di banding format docx

Nah, setelah tau kelebihan file berekstensi PDF, lalu timbul pertanyaan bagaimana merubah file berformat dokumen (doc), menjadi PDF ?

Langkah yang dapat dilakukan adalah sebagai berikut :

Buka file berekstensi dokumen yang akan Anda ubah menjadi file berekstensi PDF (dalam kasus ini penulis menggunakan aplikasi Ms. Word 2013)Buka tab file, dan pilih eksporPilih "Buat PDF/XPS"Lalu beri nama file berekstensi PDF yang Anda inginkanDan klik "terbitkan"Selamat file berekstensi dokumen Anda sudah menjadi file berformat PDF :)

*Jika Anda memiliki aplikasi pihak ke 3, seperti foxit phantom PDF, maka pengubahan file dokumen atau gambar akan menjadi lebih mudah, yakni hanya dengan klik kanan kursor pada file yang akan diubah ekstensinya, lalu pilih "convert to pdf"
