6. Pengertian network id, host id, dan broadcast id


Pengertian✍️

Network id

Network ID adalah bagian dari IP address yang menunjukkan di jaringan mana komputer tersebut berada.

Host id

Host adalah identitas dari host (interface router, workstation, dan device) yang terhubung ke jaringan.

Broadcast id

broadcast adalah alamat jaringan yang digunakan untuk mengirim ke semua perangkat yang terhubung ke jaringan komunikasi akses ganda.

Dalam hal ini teaching factory smk kompetensi keahlian multimedia akan memproduksi id card siswa kelas x dengan output barang jadi sejumlah 600 id card selama 1 bulan. berikut adalah rincian biaya produksi cetak id card tersebut selama satu bulan :biaya pembelian bahan baku = rp. 740.000biaya cetak id card = rp . 410.00biaya tenaga kerja langsung = rp. 2.400.000biaya sewa alat = rp. 250.000jika persentase keuntungan yang diharapkan adalah 40% dari biaya produksi maka berapakah harga jual tiap unit nya?


Agar diperoleh keuntungan 40%, setiap unit ID card dijual seharga Rp8.866,67. Harga ini didapat dengan konsep aritmatika sosial.

Penjelasan dengan langkah-langkah:

Hitung total biaya produksi.

Total biaya produksi = 740.000+410.000+2.400.000+250.000 = Rp3.800.000,00

Karena diharapkan keuntungan 40%, maka total penjualannya haruslah sebesar:

Total penjualan = (100%+Persentase untung)×Total biaya produksi

= (100%+40%)×3.800.000

= 140%×3.800.000

= 1,4×3.800.000

= Rp5.320.000,00

Karena output barang jadinya sebanyak 600 ID card, maka harga jual per unitnya adalah:

Harga per unit = Total penjualan/Banyaknya barang yang dijual

= 5.320.000/600

≈ Rp8.866,67

Jadi, harga jual tiap unit ID card adalah Rp8.866,67.
