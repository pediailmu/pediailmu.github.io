24. X = (A + B) / ((C - D) * E) Buatlah program untuk menghitung nilai X di atas dengan menggunakan instruksi 3-alamat, 2-alamat, 1-alamat, dan 0-alamat!


Jawaban dan Penjelasan:

Diberikan ekspresi: X = (A + B) / ((C - D) * E)

__________________________

Program INSTRUKSI 3-ALAMAT

Sintaks: [opcode] [alamat-hasil] [alamat-operand1] [alamat-operand2]

Keterangan:
⇒ R1 dan R2 adalah register.
⇒ M[ ] adalah lokasi memori tertentu.

Program:

ADD R1, A, B        ; R1 = M[A] + M[B]
SUB R2, C, D        ; R2 = M[C] – M[D]
MUL R2, R2, E      ; R2 = R2 * M[E]
DIV X, R1, R2        ; M[X] = R1 / R2

__________________________

Program INSTRUKSI 2-ALAMAT

Sintaks: [opcode] [alamat-hasil] [alamat-operand]

Keterangan:
⇒ R1 dan R2 adalah register.
⇒ M[ ] adalah lokasi memori tertentu.

Program:

MOV R1, A         ; R1 = M[A]
ADD R1, B         ; R1 = R1 + M[B]
MOV R2, C        ; R2 = M[C]
SUB R2, D         ; R2 = R2 – M[D]
MUL R2, E        ; R2 = R2 * M[E]
DIV R1, R2         ; R1 = R1 / R2
MOV X, R1         ; M[X] = R1

__________________________

Program INSTRUKSI 1-ALAMAT

Sintaks: [opcode] [alamat-operand]

Keterangan:
⇒ AC adalah register akumulator.
⇒ M[ ] adalah lokasi memori tertentu.
⇒ M[T] adalah lokasi memori sementara (temporary).

Program:

LOAD A       ; AC = M[A]
ADD B        ; AC = AC + M[B]
STORE X    ; M[X] = AC
LOAD C       ; AC = M[C]
SUB D         ; AC = AC – M[D]
MUL E         ; AC = AC * M[E]
STORE T     ; M[T] = AC
LOAD X       ; AC = M[X]
DIV T          ; AC = AC / M[T]
STORE X    ; M[X] = AC

__________________________

Program INSTRUKSI 0-ALAMAT

Sintaks: [opcode]

Notasi Postfix: X =AB+CD–E*/

Keterangan:
⇒ menggunakan stack S = [top, ... → ..., bottom]

Program:

PUSH     ; push A ke stack ⇒ S = [ A ]
PUSH     ; push B ke stack ⇒ S = [ B, A ]
ADD       ; A+B  ⇒ S = [ A+B ]
PUSH     ; push C ke stack ⇒ S = [ C, A+B ]
PUSH     ; push D ke stack ⇒ S = [ D, C, A+B ]
SUB       ; C–D  ⇒ S = [ C–D, A+B ]
PUSH   ; push E ke stack ⇒ S = [ E, C–D, A+B ]
MUL     ; (C–D)*E  ⇒ S = [ (C–D)*E, A+B ]
DIV       ; (A+B) / ((C–D)*E)  ⇒ S = [ (A+B) / ((C–D)*E) ]
POP       ; pop elemen teratas stack, X = (A+B) / ((C–D)*E)

33. Sebuah Jaringan dengan alamat : 203.153.37.0/26, Tentukan : a) Alamat Subnet Mask, b) Alamat Subnet, c) Alamat Broadcast, d) Jumlah Host yang dapat digunakan​


Jawaban:

a) Alamat Subnet Mask:

Subnet mask untuk alamat jaringan 203.153.37.0/26 adalah 255.255.255.192.

b) Alamat Subnet:

Untuk menentukan alamat subnet, kita harus melihat nilai bit di luar subnet mask. Dalam kasus ini, kita memiliki 6 bit yang tersisa untuk host (yang diberikan oleh /26). Menggunakan 6 bit ini, kita dapat memiliki 2^6 = 64 subnets.

Alamat subnet yang mungkin dalam jaringan ini adalah:

203.153.37.0

203.153.37.64

203.153.37.128

203.153.37.192

c) Alamat Broadcast:

Alamat broadcast adalah alamat terakhir dalam setiap subnet. Untuk menentukan alamat broadcast dalam subnet ini, kita mengatur semua bit host menjadi 1 (bernilai biner).

Alamat broadcast yang mungkin dalam jaringan ini adalah:

203.153.37.63

203.153.37.127

203.153.37.191

203.153.37.255

d) Jumlah Host yang dapat digunakan:

Jumlah host yang dapat digunakan dalam setiap subnet dapat dihitung dengan menggunakan rumus 2^n - 2, di mana n adalah jumlah bit host yang tersedia. Dalam kasus ini, kita memiliki 6 bit host yang tersedia.

Jadi, jumlah host yang dapat digunakan dalam setiap subnet adalah 2^6 - 2 = 62 host.
