2. turunna y=2x²-1/3x+5​


Rumus yang diberikan adalah y = 2x² - (1/3)x + 5. Rumus ini merupakan suatu persamaan kuadratik yang dapat digunakan untuk menggambarkan suatu kurva parabola.

Dalam persamaan ini, koefisien-koefisien yang terlibat adalah sebagai berikut:

- Koefisien a = 2, merupakan koefisien dari suku x² yang menentukan bentuk parabola (apakah terbuka ke atas atau terbuka ke bawah).

- Koefisien b = -1/3, merupakan koefisien dari suku x yang menentukan pergeseran horizontal parabola.

- Koefisien c = 5, merupakan konstanta yang menentukan pergeseran vertikal parabola.

Untuk menggambarkan kurva parabola, kita dapat menggunakan beberapa metode, seperti menggunakan titik-titik penting atau menggunakan teknik plotting. Berikut adalah langkah-langkah untuk menggambarkan kurva parabola dari persamaan tersebut:

1. Tentukan titik-titik penting:

a. Axis of Symmetry (sumbu simetri): x = -b / (2a)

Dalam persamaan ini, sumbu simetri dapat dihitung sebagai x = -(-1/3) / (2*2) = 1/12.

b. Vertex (titik puncak): (h, k)

Dalam persamaan ini, koordinat titik puncak dapat dihitung sebagai (1/12, f(1/12)), di mana f(x) adalah nilai y dari persamaan.

c. Intercepts (titik potong dengan sumbu x dan sumbu y):

- Untuk mencari titik potong dengan sumbu x, kita set y = 0 dan cari nilai-nilai x yang memenuhi persamaan.

- Untuk mencari titik potong dengan sumbu y, kita set x = 0 dan cari nilai y yang memenuhi persamaan.

2. Plotting titik-titik penting:

- Tandai titik puncak (h, k).

- Tandai titik potong dengan sumbu x dan sumbu y.

3. Tentukan arah lengkung parabola:

- Jika koefisien a positif (a > 0), parabola terbuka ke atas.

- Jika koefisien a negatif (a < 0), parabola terbuka ke bawah.

4. Gambar kurva parabola:

- Gunakan titik-titik penting yang telah ditentukan untuk menggambarkan kurva parabola secara halus.

Perlu diingat bahwa ini hanya langkah-langkah umum dalam menggambarkan kurva parabola. Hasil akhirnya dapat bervariasi tergantung pada koefisien dalam persamaan tersebut.

Mari cari titik potong untuk setiap pasangan fungsi yang diberikan:

a. x + y = 2 dan x - y = 2

Dari persamaan pertama, kita bisa menyatakan y = 2 - x. Kemudian, kita substitusi ini ke persamaan kedua:

x - (2 - x) = 2

x - 2 + x = 2

2x - 2 = 2

2x = 4

x = 2

Kemudian, substitusi nilai x ini ke salah satu persamaan untuk mendapatkan nilai y:

2 + y = 2

y = 2 - 2

y = 0

Jadi, titik potongnya adalah (2, 0).

Langkah-langkah serupa dapat diikuti untuk bagian lain dari pertanyaan ini dengan mencari nilai x dan y yang memenuhi kedua persamaan dalam setiap pasangan.

b. 3x + 2y = 6 dan 2x - 4y = 4

c. 2x - 5y = 15 dan 3x + 4y = 11

d. x + 2y = 4 dan 3x - y = 5

e. x + 3y = 1 dan 2x - y = 9

f. 2x1 + x2 = 8 dan x1 - x2 = 1

Anda dapat menggunakan metode substitusi atau eliminasi untuk menyelesaikan setiap sistem persamaan di atas dan mencari titik potongnya. Jika Anda ingin solusi rinci untuk salah satu pasangan tertentu, beri tahu saya, dan saya akan memberikan langkah-langkahnya.

37. Garis yang mempunyai grafik fungsi y = x + 2/2x -1 adalah a. y-3x-2 = 0 b. y + 3x - 2 = 0 c. y + 3x - 2 = 0 d. y + 3x + 2 = 0 E. y = 2x + 1


Pengembangan Materi Fungsi

Mencari hubungan antara dua fungsi dengan uji diskriminan (D = b² - 4ac)

Penyelesaian:

Siapkan fungsi [tex]y = \frac{x + 2}{2x - 1}[/tex]

Opsi (a). 
y = 3x + 2 substitusi ke dalam fungsi di atas
[tex]3x + 2 = \frac{x + 2}{2x - 1}[/tex]
[tex]6x^2 + x - 2 = x + 2[/tex]
6x² - 4 = 0
D = (0)² - 4(6)(-4) ⇒ D > 0
Kedua fungsi berpotongan (tentu di dua titik)

Opsi (b) dan (c), karena fungsi yang tertulis adalah sama (typo?)
y = -3x + 2 substitusi ke dalam fungsi di atas
[tex]-3x + 2 = \frac{x + 2}{2x - 1}[/tex]
[tex]-6x^2 + 7x - 2 = x + 2[/tex]
6x² - 6x + 4 = 0 atau 3x² - 3x + 2 = 0
D = (-3)² - 4(3)(2) ⇒ D < 0
Kedua fungsi tidak berpotongan

Opsi (d). 
y = -3x - 2 substitusi ke dalam fungsi di atas
[tex]-3x - 2 = \frac{x + 2}{2x - 1}[/tex]
[tex]-6x^2 - x + 2 = x + 2[/tex]
6x² + 2x = 0 atau 3x² + x = 0
D = (1)² - 4(3)(0) ⇒ D > 0
Kedua fungsi berpotongan (tentu di dua titik)

Opsi (e). 
y = 2x + 1 substitusi ke dalam fungsi di atas
[tex]2x + 1 = \frac{x + 2}{2x - 1}[/tex]
[tex]4x^2 - 1 = x + 2[/tex]
6x² - x - 3 = 0
D = (-1)² - 4(6)(-3) ⇒ D > 0
Kedua fungsi berpotongan (tentu di dua titik)

Hanya ada satu opsi di atas yang berbeda sendiri hasil ujinya, yaitu opsi 
y + 3x - 2 = 0.
