/**

* Program to list databases in MySQL Server using Java

public class MySQLDatabaseExample {

 

   static Connection conn = null;

   static String username = "username"; // replace with your MySQL client username

   static String password = "password"; // replace with your MySQL client password

 

   public static void main(String[] args){

       // make a connection to MySQL Server

   public static void executeMySQLQuery(){

       Statement stmt = null;

       ResultSet resultset = null;

 

       try {

           stmt = conn.createStatement();

           resultset = stmt.executeQuery("SHOW DATABASES;");

 

           if (stmt.execute("SHOW DATABASES;")) {

               resultset = stmt.getResultSet();

           }

 

           while (resultset.next()) {

               System.out.println(resultset.getString("Database"));

           }

       }

       catch (SQLException ex){

           // handle any errors

           ex.printStackTrace();

       }

       finally {

           // release resources

           if (resultset != null) {

               try {

                   resultset.close();

               } catch (SQLException sqlEx) { }

               resultset = null;

           }

 

           if (stmt != null) {

               try {

                   stmt.close();

               stmt = null;

           if (conn != null) {

               try {

               conn = null;

    * This method makes a connection to MySQL Server

    * In this example, MySQL Server is running in the local host (so 127.0.0.1)

       Properties connectionProps = new Properties();

       connectionProps.put("user", username);

           Class.forName("com.mysql.jdbc.Driver").newInstance();

                           "127.0.0.1" +

                           ":" + "3306" + "/"+

                           "",

23. macam slide layout dan fungsinya


- Teks Layouts, yang berisi Layout khusus untuk menampilkan teks saja, terdiri
dari Title Slide, Title Only, Title and Teks.

- Content Layouts, yang berisi Gambar, bagan, diagram dengan lima bentuk yaitu: Blank artinya kosong, Content berisi isinya saja, Title and Content untuk menampilkan judul dan Isinya, Title and two Content untuk menmpilkan judul dan dua isi yang disajikan dalam dua kolom, Title, Content and 2 Content sama dengan Title and Content hanya Content kolom kedua dipecah lagi menjadi dua baris.
 
- Text and Contents Layout
Yaitu gabungan antara teks dengan gambar, bagan atau diagram, anda dapat memilih salah satu model yang diinginkan.

- Other Layout
Model Slide out yang lain, misalnya dengan clip art, media dan sebagainya. Anda dapat memilih sesuai dengan kebutuhan

32. apa fungsi dari print layout?


Kelas: X

Mata pelajaran: Teknologi Informasi

Materi: Microsoft Word

 

Saya akan mencoba menjawab dengan dua jawaban:

 

Jawaban pendek:


Print layout berguna menampilkan dokumen pada Microsoft WOrd sebagaimana akan tampak dokumen tersebut ketika dicetak.  

Jawaban panjang:  

Dalam Microsoft Word ada lima mode untuk melihat dokumen yaitu:  

Read Mode  

Dokumen mengisi jendela Microsoft Word dan diformat dalam bentuk seperti tampilan di tablet atau majalah. Tampilan ini sangat berguna untuk memeriksa dokumen, serta membaca yang telah Anda download atau terima di email.  

Print Layout  

Tampilan umum Microsoft Word. Dokumen umum, seperti memo, surat, dan laporan, sering ditulis dan diedit dalam tampilan Print Layout. Salah satu keuntungan bekerja dalam pandangan ini adalah kepatuhannya terhadap WYSIWYG (apa yang Anda lihat adalah apa yang Anda dapatkan). Margin, header / footer, dan format sesuai dengan hasil cetak. Halaman ditampilkan sebagai potongan kertas setara dengan jeda fisik antar halaman.  

Web Layout  

Tampilan Web Layout digunakan untuk membuat, melihat, dan mengedit halaman karena akan muncul secara online saat dibuka di browser. Dengan memilih Save As di mode ini, kitadapat menyimpan halaman dalam beberapa format yang kompatibel dengan web.  

Outline  

Outline akan menampilkan susunan dokumen seperti susunan bab, sub bab dan point penting dokumen.  

 Draft

 

Draft digunakan saat kecepatan sangat penting, misalnya saat menggunakan komputer yang lambat. Tampilan Draft menampilkan dokumen sebagai teks kontinu. Jeda halaman dilambangkan dengan garis putus-putus.
