Media penyimpanan sekunder adalah media penyimpanan yang dapat menyimpan data lebih banyak, dari media penyimpanan primer.

Fungsinya adalah:

Menyimpan data dengan lebih banyak daripada media penyimpanan primer.Menyimpan data bisa secara permanen, tidak seperti media penyimpanan primer pada umumnya.Menyimpan data atau program, yang belum di execute oleh processor, sehingga meringankan tugas processor.

______________________________________________________

PenjelasanPengertian media penyimpanan:

Media penyimpanan pada komputer adalah menyimpan berbagai macam data berbentuk digital yang terdapat pada perangkat komputer, agar dapat dibuka, dan diakses kembali pada suatu saat pada perangkat komputer.

Jenis media penyimpanan:Media penyimpanan primerMedia penyimpanan sekunder

Media penyimpanan primer yaitu:

Media penyimpanan primer pada komputer ialah penyimpanan yang menyimpan yang menyimpan data dalam hardware berbentuk RAM (Random Access Memory) dan Rom (Read Only Memory). Data yang disimpan adalah data yang baru dibuka atau dieksekusi oleh perangkat komputer. Data yang sudah diinput ke RAM/ROM akan terhapus jika tidak diakses dalam jangka waktu tertentu.

Contoh: RAM, dan ROM.

Media penyimpanan sekunder yaitu:

Media penyimpanan sekunder pada komputer ialah penyimpanan yang menyimpan data dalam hardware berbentuk Hard Disk, CD/DVD, Floppy Disk, dsb. Data yang disampan dalam penyimpanan sekunder adalah data yang disimpan secara permanen, dan tidak akan terhapus seperti RAM/ROM.

Contoh: Hard Disk, CD, DVD, Floppy Disk, Flash Disk, Mobile storage.

1.Input Storage: Untuk menampung data yang dibaca.

2.Program Storage Area :Penyimpanan intruksi - instruksi untuk pengolahan.

Sederhananya fungsi dari atribut "alt" pada elemen atau tag image <img> adalah sebagai teks alternatif; artibut "width" dah "height" berfungsi untuk mendeklarasikan ukuran dari elemen image <img> (gambar); kedua atribut (width dan height) tersebut harus selalu digunakan secara bersamaan pada sebuah gambar; Cara membuat hyperlink atau tautan dari sebuah gambar bisa dilakukan dengan mengapit tag <img> dengan tag <a> dimana gambar menjadi jangkarnya (anchor)

Pembahasan:

ALT

Atribut alt pada image adalah singkatan dari alternative text yang pada dasarnya digunakan untuk membuat alternatif teks dari sebuah gambar yang berasal dari elemen <img>. Bagi pengguna pada umumnya fungsi alt ini tidak terlalu signifikan karena pada dasarnya ditujukan untuk:

peramban atau browser agar menampilkan teks ketika halaman dimana gambar tersebut berada gagal loading dimana gambar tidak bisa ditampilkan, misalnya ketika ada galat (error) atau koneksi internet yang tidak mencukupi-keperluan komputasi agar peramban dan browser bisa membaca teks pada alt untuk dirubah menjadi suara. Biasanya fasilitas ini digunakan oleh pengguna disabilitas yang memiliki gangguan pengelihatan (tuna netra)  dibaca oleh bot mesin penelusur agar mesin pencari bisa menyimpan gambar tersebut sesuai dengan konteksnya dimana nantinya bisa ditampilkan pada halaman pencarian yang tepat sesuai kata kunci para penggunanya

Width dan Height

Dalam generasi HTML yang ada saat ini width dan height hanya bisa digunakan pada elemen <img>. Fungsinya sendiri adalag agar mempercepat peramban untuk menampilkan gambar dalam ukuran aslinya. Agar tidak ada ketimpangan ukuran pada saat proses loading width dan height harus diaplikasikan secara bersamaan.

Tautan Image

Image atau gambar bisa dijadikan tautan, caranya hampir sama ketika kita membuat tautan teks hanya saja pada gambar yang dijadikan jangkarnya adalah image tersebut. Berikut contohnya:

<img alt="isi dengan alternatif teks gambar" src="contohsumbergambar.com/google.jpg">

</a>

12. 1. Jelaskan apa yang dimaksud dengan parameter MYSQLI_ASSOC , MYSQLI_NUM, MYSQLI_BOTH !2. Jelaskan dan tuliskan parameter yang digunakan dalam fungsi mysqli_fetch_rows !


Yang dimaksud dengan parameter MYSQLI_ASSOC, MYSQLI_NUM, dan MYSQLI_BOTH adalah parameter yang digunakan dalam bentuk array. Perbedaan ketiga parameter tersebut hanya terletak pada cara array tersebut didefinisikan.


Pada MYSQLI_NUM = Item array akan menggunakan indeks key bertipe numerik.

Pada MYSQLI_ASSOC = Item array akan menggunakan nama kolom sebagai index key

Pada MYSQLI_BOTH = Item array akan diduplikasikan, yang satu menggunakan indeks key bertipe numerik, dan yang lainnya menggunakan nama kolom sebagai index key


Parameter yang digunakan dalam fungsi mysqli_fetch_rows adalah sebagai berikut

mysqli_fetch_row(result);


Result wajib diisi. Menandakan kumpulan hasil identifier yang dikembalikan oleh mysqli_query(), mysqli_store_result() atau mysqli_use_result().


mysqli_fetch_rows digunakan untuk mengambil sebuah baris dari kumpulan hasil dan mengembalikannya ke dalam bentuk array yang telah dicacah

23. Apa kekurangan dan kelebihan media penyimpanan primer dan sekunder


Kekurangan dan kelebihan media penyimpanan primer dan sekunder adalah :

1.     Media Primer

Kelebihan :

-  Memiliki kecepatan akses yang lebih cepat

-  Penyimpanan terhubung langsung dengan processing device

-  Lebih cocok untuk pemrosesan data dari pada penyimpanan

Kekurangan :

-  Memiliki harga yang lebih mahal daripada penyimpanan sekunder

-  Bukan penyimpanan portable yang bisa dibawa seperti sekunder (       flashdisk, harddisk eksternal dll. )

-  Kapasitas lebih terbatas dan cenderung kecil

-  Memiliki resiko lebih tinggi untuk terjadi kehilangan data jika terjadi situasi powercut mendadak

2.   Media Sekunder

Kelebihan :

-  Memiliki harga yang lebih terjangkau daripada penyimpanan primer

-  Sifatnya portable sehingga dapat dipindah - pindahkan

-  Kapasitas besar dan beragam

-  Lebih aman dalam penyimpanan data jika terjadi situasi powercut

Kekurangan :

-   Memiliki kecepatan akses data yang lambat

-   Penyimpanan tidak terkoneksi dengan pemroses

-   Hanya dapat berperan sebagai penyimpanan data saja.

Pembahasan

Media penyimpanan primer adalah media penyimpanan yang terletak didalam komputer dan memiliki koneksi langsung dengan pemroses sehingga menunjang proses data yang bersifat temporary. Karena sifatnya yang temporary dalam menyimpan data, jika terjadi situasi powercut, data akan hilang

Contoh :

RAM  ROM

Media penyimpanan sekunder adalah media penyimpanan yang terletak tidak didalam komputer dan cenderung portable. Lebih berfungsi sebagai penyimpanan data sehinggansekalipun dalam situasi powercut data akan tetap aman.

Contoh :

HDD  dan SSDFlashdriveCD-RW

38. Apa perbedaan dari media penyimpanan utama dengan media penyimpanan sekunder?


Penyimpanan utama (contohnya RAM), ialah penyimpanan data dan program yang bersifat sementara.
Dalam pengertian: data dan program tersebut akan hilang ketika komputer dimatikan.

Penyimpanan Sekunder (penyimpanan pembantu) ialah  beberapa alat penyimpanan yang di desain untuk menahan data atau program dalam  bentuk relatif permanen (contohnya CD)
Dalam pengertian bahwa penyimpanan data dan program tetap ada walaupun komputer dimatikan.
