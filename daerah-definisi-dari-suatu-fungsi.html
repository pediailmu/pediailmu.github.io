Jawaban:

Agar fungsi f(x) = √3 – x : √x² + 3x – 10 terdefinisi dalam daerah asalnya, maka daerah asal fungsi f adalah Df = {x| x < –5 atau 2 < x ≤ 3, x ∈ R}. Untuk menentukan daerah asal dari suatu fungsi, kita lihat bentuk fungsinya bagaimana.

y = \sqrt{f(x)}

f(x)

⇒ Df : f(x) ≥ 0

y = \frac{f(x)}{g(x)}

g(x)

f(x)

⇒ Df : g(x) ≠ 0

y = \frac{f(x)}{\sqrt{g(x)}}

g(x)

f(x)

⇒ Df : g(x) > 0

Df = domain atau daerah asal fungsi

Pembahasan

f(x) = \frac{\sqrt{3 - x}}{\sqrt{x^{2} + 3x - 10}}

x

2

+3x−10

3−x

Daerah asal pembilang: angka didalam akar tidak boleh negatif, maka

3 – x ≥ 0

3 ≥ x

x ≤ 3

Daerah asal penyebut: angka didalam akar tidak boleh negatif dan penyebut tidak boleh nol, maka

x² + 3x – 10 > 0

(x + 5)(x – 2) > 0

x = –5 atau x = 2

Garis bilangan:

++++ (–5) ------ (2) ++++++
