beberapa saran yang dapat disampaikan, yaitu :

rancangan ini dapat dikembangkan dengan lebih baik.

12. apa perbedaan web server, DNS server dan FTP server ?


Jawaban:

Web Server

Web Server dapat merujuk baik pada perangkat keras ataupun perangkat lunak yang menyediakan layanan akses kepada pengguna melalui protokol komunikasi HTTP atau HTTPS atas berkas-berkas yang terdapat pada suatu situs web dalam layanan ke pengguna dengan menggunakan aplikasi tertentu seperti peramban web.

DNS Server

DNS (Domain Name System) sendiri adalah sebuah sistem yang menyimpan informasi tentang nama host ataupun nama domain dalam bentuk basis data tersebar (distributed database) di dalam jaringan komputer, misalkan: Internet. DNS menyediakan alamat IP untuk setiap nama host dan mendata setiap server transmisi surat (mail exchange server) yang menerima surel (email) untuk setiap domain. Menurut browser Google Chrome, DNS adalah layanan jaringan yang menerjemahkan nama situs web menjadi alamat internet.

FTP Server

FTP Server adalah protokol internet yang digunakan untuk urusan pengiriman data dalam jaringan komputer, seperti upload dan download file yang dilakukan oleh FTP client dan FTP server.

17. apa yang dimaksud dengan dns dan apa perbedaan public dns dengan dns default


DNS yaitu Sebuah sistem yang menyimpan informasi tentang nama host ataupun nama domain dalam bentuk basis data tersebar (distributed database) di dalam jaringan komputer.

Jawaban:

DNS atau Domain Name System berfungsi sebagai penerjemah/pengubah dari alamat situs ke bentuk IP address.

Simplenya DNS adalah buku telepon agar perangkat kita dapat menemukan IP address dari suatu situs web dengan lebih cepat.

Contoh DNS

Beberapa DNS publik yang sudah sangat umum yaitu DNS Google (8.8.8.8) dan OpenDNS (208.67.222.222). Banyak penyedia Public DNS lainnya yang memberikan layanan DNS secara gratis seperti Cloudflare, AdGuard, Quad9, Verisign, dll. yang menyediakan keunggulannya masing-masing.

Semua tergantung User dalam memilih ingin menggunakan DNS yang mana.

Perbedaan

DNS Publik atau Public DNS memiliki beberapa keunggulan tersendiri tergantung dari penyedia DNS-nya. Terkadang menggunakan Public DNS jauh lebih menguntungkan dalam beberapa hal dibandingkan menggunakan DNS default bawaan provider.

Contoh Penggunaan

Menggunakan Public DNS seperti Google DNS lebih menguntungkan daripada menggunakan DNS bawaan provider karena server Google sendiri sudah sangat canggih, jauh lebih maju dibanding server yang digunakan provider internet tertentu sehingga dalam beberapa kasus jika kita menggunakan Google DNS atau DNS publik lainnya maka aktivitas browsing kita bisa jauh lebih lancar dan cepat.

Lalu biasanya DNS bawaan dari provider internet kita sudah dimodifikasi agar menampilkan iklan yang mempromosikan produk & layanan mereka. Beberapa pengguna pastinya merasa terganggu dengan iklan-iklan ini. Hal ini tentunya tidak ada jika menggunakan Public DNS dari Google, OpenDNS, Cloudflare, dsb.

Catatan

Yang perlu diperhatikan adalah pastikan untuk selalu menggunakan Public DNS dari penyedia yang memang terpercaya seperti Google, Cloudflare, OpenDNS, dan sejenisnya. Hal ini karena aktivitas browsing dapat dicatat oleh server DNS. Agar tidak disalah-gunakan maka harus menggunakan penyedia Public DNS yang terpercaya.

23. bagaimana cara kerja dns dan dns server


DNS bekerja dengan menggunakan protokol UDP.Tujuanya adalah untuk performance.DNS pada saat client melakukan request terhadap DNS tujuan, yang pertama dilakukan mengecek pada komputer apakah request sudah ada di cache atau belum, jika ada buka, jika tidak kirim ke server, yang dikirimkan oleh Client, menggunakan query yang dikirim, setelah itu akan ada balasan dan diakses oleh browser.

Pada saat DNS Server menerima request, akan mencari domain (resolve) terhadap cache nya, jika sudah ada maka kembalikan ke client, jika tidak ada dicari ke server lain, namun jika pada detik tertentu, akan kemungkinan browser menyatakan timeout karena server terlalu lama merespon.

24. Jelaskan keterkaitan antara dns server, web server dan database server


Jawaban:

Keterkaitan diantara ketiganya adalah bahwa DNS, Web server, dan Database server sama-sama dibutuhkan dalam menghosting sebuah website yang dynamic.

Penjelasan:

DNS = Domain Name Server, berfungsi untuk mengubah alamat ip kita menjadi sebuah nama yang kita inginkan dan memudahkan orang lain dalam mengakses website kita.

Web server = Sebuah server perangkat lunak yang berfungsi untuk melayani permintaan(request) klien berupa HTTP atau HTTPS dan akan mengirimkan respon sesuai yang diminta.

Database server = Sebuah program komputer yang menyediakan layanan pengelolaan basis data dan melayani sebuah komputer yang kemudian disalurkan ke komputer lainnya menggunakan klien-server

DNS diperlukan untuk memudahkan klien dalam mengakses website kita. Web server dibutuhkan agar terjadi request dan respon antara klien dan server. Dan database dibutuhkan untuk pengelolaan data di sebuah website kita yang menjadikannya dynamic(berubah-ubah).
