7. buatlah algoritma untuk mencari nilai terbesar dari 4 buah bilangan di inputkan dari keyboard​


Berikut ini adalah algoritma untuk mencari nilai terbesar dari empat buah bilangan yang diinputkan dari keyboard:

MulaiBaca bilangan pertamaBaca bilangan keduaBaca bilangan ketigaBaca bilangan keempatBandingkan bilangan pertama dengan bilangan kedua, jika bilangan pertama lebih besar dari bilangan kedua, maka lanjutkan ke langkah selanjutnya, jika tidak, maka tukar nilai bilangan pertama dengan bilangan keduaBandingkan bilangan pertama dengan bilangan ketiga, jika bilangan pertama lebih besar dari bilangan ketiga, maka lanjutkan ke langkah selanjutnya, jika tidak, maka tukar nilai bilangan pertama dengan bilangan ketigaBandingkan bilangan pertama dengan bilangan keempat, jika bilangan pertama lebih besar dari bilangan keempat, maka lanjutkan ke langkah selanjutnya, jika tidak, maka tukar nilai bilangan pertama dengan bilangan keempatCetak bilangan pertamaSelesai

Berikut ini adalah algoritma untuk mencari nilai terbesar dari 4 buah bilangan yang diinputkan dari keyboard:

Mulai

Buat variabel A, B, C, dan D untuk menyimpan 4 buah bilangan yang diinputkan

Baca input untuk A, B, C, dan D

Buat variabel MAX untuk menyimpan nilai terbesar

Jika A > MAX, maka set MAX = A

Jika B > MAX, maka set MAX = B

Jika C > MAX, maka set MAX = C

Jika D > MAX, maka set MAX = D

Tampilkan MAX

Selesai.

16. fungsi keyboard, mouse, monitor, cpu cari dari Google pakai gambar​


Jawaban:

Berikut adalah fungsi dari setiap perangkat keras dalam sebuah komputer:

1. Keyboard:

Fungsi utama keyboard adalah sebagai perangkat input untuk memasukkan teks, angka, dan simbol ke dalam komputer. Dengan keyboard, pengguna dapat mengetikkan dokumen, mengirim pesan, memasukkan perintah, dan berinteraksi dengan berbagai aplikasi dan program.

2. Mouse:

Fungsi utama mouse adalah sebagai perangkat input yang mengontrol gerakan kursor pada layar monitor. Mouse memungkinkan pengguna untuk memilih, mengklik, menggeser, dan berinteraksi dengan berbagai elemen pada layar seperti ikon, menu, teks, dan aplikasi.

3. Monitor:

Fungsi utama monitor adalah sebagai perangkat output yang menampilkan visual dari komputer. Dengan monitor, pengguna dapat melihat informasi, gambar, video, dan tampilan grafis yang dihasilkan oleh komputer.

4. CPU (Central Processing Unit):

CPU merupakan otak atau pusat pengendalian dari komputer. Fungsi utama CPU adalah untuk menjalankan instruksi dan perhitungan yang diberikan oleh perangkat lunak (software) dan mengkoordinasi berbagai komponen komputer. CPU bertanggung jawab atas pengolahan data, menjalankan program, dan mengendalikan operasi keseluruhan komputer.

Secara keseluruhan, keyboard dan mouse berfungsi sebagai perangkat input untuk memasukkan perintah dan berinteraksi dengan komputer, sedangkan monitor dan CPU berfungsi sebagai perangkat output dan pusat pengendalian yang menampilkan informasi dan menjalankan operasi komputer secara keseluruhan. Semua perangkat ini bekerja bersama-sama untuk menyediakan pengalaman pengguna dalam menggunakan komputer dengan cara yang efisien dan efektif.

 Fungsi keyboard adalah mengirimkan perintah dari tiap huruf maupun angka yang kita ketikkan kemudian menyampaikannya ke CPU dan selanjutnya perintah manual tersebut akan di rubah menjadi sinyal-sinyal digital sehingga dapat dimengerti oleh prosessor.

FUNSI MONITOR
Monitor berfungsi sebagai Output dari memori komputer atau central processing unit berupa biner. Ini harus diubah menjadi bahasa manusia dan ditampilkan kemonitor sehingga bisa dibaca oleh pengguna.

FUNGSI CPU
Fungsi utama dari CPU adalah melakukan operasi aritmetika dan logika terhadap data yang diambil dari memori atau dari informasi yang dimasukkan melalui beberapa perangkat keras, seperti papan tombol, pemindai, tuas kontrol, maupun tetikus.

36. cari artikel dgn judul fungsi dan kegunaan pada keyboard​


Jawaban:

1. Tombol ESC

Fungsi tombol esc adalah untuk menghentikan, membatalkan atau keluar dari aplikasi secara cepat. Tombol esc berfungsi untuk membatalkan sebuah perintah atau menutup aplikasi. Tombol esc pada jenis perangkat apapun berada di ujung kiri paling atas.

2. Tombol Tab

Fungsi tombol tab adalah untuk membuat kursor bergeser dengan jarak 1 tabulasi ke arah kanan. Fungsi tombol ini mirip dengan tombol space yang digunakan untuk memberi jarak antara kata 1 dan lainnya.  Letak tombol tab tepat diatas tombol caps lock.

3. Tombol Caps Lock

Fungsi tombol caps lock adalah untuk membuat huruf atau tulisan menjadi BESAR (UPPERCASE). Biasanya ada indikator atau lampu yang menyala begitu tombol ini dalam keadaan on.

4. Tombol Windows (Home)

Fungsi tombol home adalah untuk mengarahkan tampilan pada halaman awal atau halaman start secara cepat. Untuk mengembalikan pada halaman yang terakhir dibuka, maka cukup klik lagi tombol ini. Tombol home biasanya berupa logo windows atau jendela dan terletak antara CTRL dan ALT.

5. Tombol Space (Spasi)

Tombol space berbentuk panjang dan terletak dibagian paling bawah. Fungsinya adalah untuk menambahkan spasi atau jarak antara 1 kata dengan kata lainnya.

6. Tombol Delete

Tombol delete berfungsi untuk menghapus 1 karakter di kanan/depan kursor. Tombol ini akan menghapus kata yang ditulis setelah tanda kursos atau kata yang sebelumnya telah ditandai.

7. Tombol Enter

Fungsi tombol enter adalah untuk berpindah baris. Ketika mengetikan tombol enter maka tulisan di baris kursor akan berpindah ke baris baru. Tombol enter ada disebelah kanan keyboard dengan bentuk paling besar diantara tombol lainnya.

8. Tombol Backspace

Tombol Backspace ditandai dengan simbol <—- di keyboard. Fungsi tombol backspace adalah untuk menghapus satu karakter yang ditulis. Jika kamu ingin menghapus lebih cepat gunakan perpaduan tombol backspace dengan CTRL untuk menghapus sebuah kata.

9. Tombol PrtScr

Tombol print screen atau pada keyboard disingkat PrtScr/PrtSc adalah tombol yang digunakan untuk menangkap atau meng-capture layar. Tekan tombol ini 1 kali kemudian buka ms word ataupun paint. Ketik CTRL + P, maka gambar hasil screeshoot pun akan muncul.

Cara lain adalah dengan memadukan tombol windows/home dengan printscreen. Ketika di klik berbarengan maka akan muncul kedipan di layar yang menandakan layar sudah ter-screenshoot. Kamu dapat melihat hasilnya di folder PICTURES di my computer anda.

10. Tombol End

Fungsi tombol end adalah untuk memindahkan kursor ke baris akhir tulisan. Ini biasanya berjalan ketika anda sedang bekerja menggunakan ms word.

11. Tombol Page Up & Page Down

Tombol page up digunakan untuk menarik halaman ke atas (scrolling ke atas), dan page down untuk menarik halaman ke bawah (scrolling ke bawah). Dengan menggunakan tombol ini anda tak perlu repot ketika akan menurunkan atau menaikan posisi file di layar komputer.

12. Tombol Navigasi (Panah Kanan, kiri, atas, bawah)

Tombol navigasi berfungsi untuk menggerakan kursor ke atas, bawah, kanan ataupun kiri. Kursor sendiri adalah bagian di layar komputer yang berfungsi sebagai indikator posisi. Bentuknya lurus kecil dan berkedip-kedip.

   Tombol Scrooll Lock -> Pada microsoft excel tombol ini berfungsi untuk mengunci kolom agar tidak menggulung saat  tombol atas kanan kiri bawah ditekan.
