1) SUM = Untuk menjumlahkan sekumpulan data pada suatu range.

2) AVERAGE = Untuk menemukan nilan rata-rata dari suatu range.

3) AND = Fungsi AND adalah kebalikan dari fungsi OR. Kegunaannya sama yaitu untuk membandingkan 2 atau lebih kondisi dan menghasilkan nilai TRUE atau FALSE. 4) LEFT = Untuk mengambil beberapa karakter text dari data di Microsoft Excel yang dimulai dari kiri.1. SUM, yaitu fungsi untuk menjumlahkan isi sel dalam suatu range
2. AVERAGE, yaitu fungsi untuk menghitung nilai rata-rata dari suatu range data
3. AND, yaitu fungsi untuk menampilkan nilai boolean, yaitu TRUE apabila kedua sel bernilai benar dan FALSE untuk selainnya
4. LEFT, yaitu fungsi untuk mengambil karakter dalam suatu sel dari sebelah kiri sebanyak n karakter

26. Jelaskan kegunaan fungsi logika dari: IF; AND; OR; dan NOT(masing-masing contohnya)


Logika IF : memberikan pernyataan suatu kondisi.
Contoh : "=IF(B5>75,"Lulus",Gagal")"

Logika NOT : membalikkan suatu pernyataan dengan output TRUE atau FALSE.
Contoh = "=NOT(50 > 70)" Maka akan bernilai TRUE(kebalikan dari False)

Logika AND : memberikan nilai TRUE jika semua inputannya bernilai benar. Jika salah satu inputannya salah, maka menghasilkan False
Contoh = B5 = 80 maka "=AND(B5>1, B5<85)"

Logika OR : Memberikan nilai TRUE jika salah satu inputannya bernilai benar. Jika kedua inputan salah, maka bernilai False. 
Contoh = B1 = 25 dan B2 = 10 "=OR(B1>10, B2>15)"

40. Jelaskan kegunaan fungsi logika dari: IF; AND; OR; dan NOT(masing-masing contohnya)


Jawaban:

Semoga membantu

Penjelasan:

Logika IF : memberikan pernyataan suatu kondisi.

Contoh : "=IF(B5>75,"Lulus",Gagal")"

Logika AND : memberikan nilai TRUE jika semua inputannya bernilai benar. Jika salah satu inputannya salah, maka menghasilkan False

Contoh = B5 = 80 maka "=AND(B5>1, B5<85)"

Logika OR : Memberikan nilai TRUE jika salah satu inputannya bernilai benar. Jika kedua inputan salah, maka bernilai False.  

Contoh = B1 = 25 dan B2 = 10 "=OR(B1>10, B2>15)"

Logika NOT : membalikkan suatu pernyataan dengan output TRUE atau FALSE.

Contoh = "=NOT(50 > 70)" Maka akan bernilai TRUE(kebalikan dari False)
