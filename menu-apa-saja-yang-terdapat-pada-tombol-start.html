15. Tuliskan langkah-langkah mengaktifkan menu dan ikon pada menu bar melalui tombol start!


Toolbar merupakan kumpulan dari ikon-ikon yang digunakan untuk mempermudah dan mempercepat menangani pekerjaan yang menggunakan program aplikasi seperti Microsoft Word, Microsoft Excel dan sebagainya.
Pada Microsoft Excel terdapat beberapa toolbar, yaitu :
1. Standard Toolbar

2. Formatting Toolbar

3. Drawing Toolbar

Jika ketiga toolbar tersebut salah satu atau lebih belum muncul di tampilan lembar kerja Microsoft Excel. Cara memunculkannya sebagai berikut :
1. Klin menu view
2. Pilih dan klik Toolbars
3. Tandai(Checklist) di depan toolbar yang ingin ditampilkan.
jika kita ingin menggunakan microsoft office , langkah pertama klik tombol start , kedua pilih microsoft office , lalu pilih microsoft apa yg ingin kita gunakan

16. jelaskan macam-macam pilihan menu yang terdapat pada menu start?


Move : digunakan untuk memindahkan jendela dari satu lokasi ke lokasi lain.

Size : untuk mengubah ukuran jendela dari besar menjadi kecil atau sebaliknya.

Minimize : untuk meminimalkan ukuran jendela menjadi sebuah ikon

Maximize : untuk memperlebar ukuran jendela sampai menyerupai atau berbentuk desktop
