Jawaban:

a. ECHO: Menampilkan pesan, atau mengubah command echoing on atau off.

b. COLOR: Mengubah warna teks dan latar pada command prompt

c. VER: Menampilkan versi sistem operasi yang digunakan

d. CLS: Membersihkan layar dan menempatkan kursor pada pojok kiri layar. Perintah ini tidak memiliki parameter.

e. PROMPT: Mengubah tampilan command prompt MS-DOS

f. DATE: Perintah ini akan menampilkan tanggal saat ini. Perintah ini juga dapat mengeset tanggal komputer.

g. LABEL: Berfungsi untuk menampilkan label atau merubah nama label pada disk.

h. COPY CON: untuk membuat file

i. TYPE: Menampilkan isi dari sebuah berkas (dalam bentuk teks) ke dalam standard output.

j. DIR: Jika digunakan tanpa parameter, perintah ini dapat menampilkan daftar berkas-berkas dan subdirektori yang terdapat di dalam direktori aktif. Berkas ini memiliki satu parameter, yakni lokasi direktori di mana hendak menampilkan daftar isi direktori.

k. COPY: Digunakan untuk meng-copy isi file yang satu ke file yang lain.

l. CHDIR/CD: Mengganti direktori aktif ke direktori lainnya yang ditentukan dalam parameter. Jika dijalankan tanpa parameter, maka perintah ini akan menampilkan lokasi di mana direktori aktif berada.

m. REN/RENAME: Mengubah nama sebuah atau beberapa berkas (dengan menggunakan karakter wildcard).

n. MOVE: Berfungsi untuk memindahkan suatu file ke lokasi lain yang berbeda.

o. DEL/ERASE: Menghapus berkas yang ditentukan dalam parameter. Parameter dapat berupa nama berkas atau beberapa nama berkas yang disusun menggunakan karakter wildcard.

p. VOL: Menampilkan nama label dari sebuah volume atau partisi.

q. ATTRIB: Menampilkan atau mengubah atribut suatu file.

a. XCOPY: Menyalin file serta pohon direktori.

b. FORMAT: Mem-Format sebuah hardisk/disket untuk digunakan di Windows.

c. CHKDSK: Berfungsi untuk melihat kondisi media penyimpanan yang kita pakai.

d. SCANDISK: untuk pemeriksaan dan perbaikan sistem berkas dan buruk Cluster pada Hard drive.

e. DEFRAG: untuk mengoptimalkan kinerja disk oleh reorganisasi file pada disk dengan mencari bad sector, dan mengompimalkan kinerja hardisk.

f. DISKCOPY: Menyalin seluruh isi disket sama persis ke disket lainnya.

g. FDISK: untuk mengkonfigurasikan partisi suatu hardisk untuk MS-DOS
