Konsep dari nilai mutlak ini adalah jika terdapat |x| maka nilainya akan menjadi |x| = x jika x ≥ 0 atau menjadi |x| = -x jika x < 0. Konsep inilai yang akan dipakai untuk menjawab soal kali ini

Nomor a

Diperhatikan |x + 1|

|x + 1| = x + 1, untuk x + 1 ≥ 0 atau x ≥ -1

Atau

|x + 1| = -(x + 1) = -x - 1, untuk x + 1 < atau x < -1

Selanjutnya untuk grafik dari y = |x + 1| adalah gabungan dari grafik y = x + 1 untuk x ≥ -1 dan grafik y = -x - 1, untuk x < -1 seperti gambar terlampir di bawah

Nomor b

Diperhatikan |x - 1|

|x - 1| = x - 1, untuk x - 1 ≥ 0 atau x ≥ 1

Atau

|x - 1| = -(x - 1) = -x + 1, untuk x - 1 < 0 atau x < 1

Jadi y = |x - 1| + 2 bisa dibreakdown menjadi 2 kondisi, yaitu

y = |x - 1| + 2 = x - 1 + 2 = x - 1, untuk x ≥ 1

Atau

y = |x - 1| + 2 = -x + 1 + 2 = -x + 3, untuk x < 1

Selanjutnya untuk grafik y = |x - 1| + 2 adalah gabungan dari grafik y = x + 1, untuk x ≥ 1 dan grafik y = -x + 3, untuk x < 1 seperti gambar terlampir di bawah

Nomor c

Diperhatikan |x - 1|

|x - 1| = x - 1, untuk x - 1 ≥ 0 atau x ≥ 1

Atau

|x - 1| = -(x - 1) = -x + 1, untuk x - 1 < 0 atau x < 1

Sehingga y = 2|x - 1| + 1 bisa dibreakdown menjadi 2 kondisi, yaitu

y = 2|x - 1| + 1 = 2(x - 1) + 1 = 2x - 2 + 1 = 2x - 1, untuk x ≥ 1

atau

y = 2|x - 1| + 1 = 2(-x + 1) + 1 = -2x + 2 + 1 = -2x + 3, untuk x < 1

Selanjutnya untuk grafik y = 2|x - 1| + 1 adalah gabungan dari grafik y = 2x - 1, untuk x ≥ 1 dan grafik y = -2x + 3, untuk x < 1 seperti gambar terlampir di bawah

**Jawaban:**

**a. Arah membuka parabola**

Arah membuka parabola ditentukan oleh nilai koefisien kuadrat a. Jika a > 0, maka parabola akan membuka ke atas. Jika a < 0, maka parabola akan membuka ke bawah.

Pada fungsi f(x) = x² + 2x - 3, nilai koefisien kuadrat a = 1. Karena a > 0, maka parabola akan membuka ke atas.

**b. Persamaan sumbu simetrinya**

Persamaan sumbu simetri parabola adalah x = -b/2a.

Pada fungsi f(x) = x² + 2x - 3, nilai b = 2 dan a = 1.

```

x = -2/2 * 1

```

```

x = -1

```

Jadi, persamaan sumbu simetrinya adalah x = -1.

**c. Koordinat titik baliknya**

Titik balik parabola terletak di sumbu simetrinya. Oleh karena itu, koordinat titik baliknya adalah (-1, f(-1)).

```

f(-1) = (-1)² + 2 * -1 - 3

```

```

f(-1) = -5

```

Jadi, koordinat titik baliknya adalah (-1, -5).

**d. Nilai balik fungsinya (maksimum/minimum)**

Nilai balik fungsi kuadrat adalah nilai fungsi pada titik baliknya. Pada fungsi f(x) = x² + 2x - 3, nilai baliknya adalah -5.

Karena a > 0, maka nilai balik fungsi adalah minimum.

**e. Koordinat titik potong parabola dengan sumbu Y**

Titik potong parabola dengan sumbu Y adalah titik yang terletak pada sumbu Y, yaitu x = 0.

```

```

```

f(0) = -3

```

Jadi, koordinat titik potongnya adalah (0, -3).

**f. Nilai p jika titik (1, p) terletak pada parabola**

Karena titik (1, p) terletak pada parabola, maka nilai f(1) = p.

```

p = -1

```

Jadi, nilai p adalah **-1**.
