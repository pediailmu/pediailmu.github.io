Fungsi [tex]\displaystyle{f(x)=\left ( \frac{3-\sqrt{x^2+5}}{4-x^2} \right )^{-1}}[/tex] tidak memiliki titik diskontinu atau fungsi kontinu di semua titik.

PEMBAHASAN

Suatu fungsi f(x) akan tidak kontinu di titik x = c jika :

1. Tidak memiliki nilai di x =c atau f(c) tidak ada.

2. Tidak memliki nilai limit di x = c.

3. Nilai fungsi dan nilai limit fungsi di x = c tidak sama.

Contoh kekontinuan beberapa fungsi, antara lain :

1. Fungsi polinom [tex]f(x)=a_1x^n+a_2x^{n-1}+...+a_0[/tex] kontinu di semua titik.

2. Fungsi rasional [tex]\displaystyle{f(x)=\frac{g(x)}{h(x)} }[/tex] kontinu di semua titik kecuali di h(x) = 0.

3. Fungsi akar [tex]f(x)=\sqrt{g(x)}[/tex] kontinu di [tex]g(x)\geq 0[/tex].

.

DIKETAHUI

[tex]\displaystyle{f(x)=\left ( \frac{3-\sqrt{x^2+5}}{4-x^2} \right )^{-1}}[/tex]

.

DITANYA

Tentukan titik f(x) diskontinu.

.

PENYELESAIAN

[tex]\displaystyle{f(x)=\left ( \frac{3-\sqrt{x^2+5}}{4-x^2} \right )^{-1}}[/tex]

[tex]\displaystyle{f(x)=\frac{4-x^2}{3-\sqrt{x^2+5}}\times\frac{3+\sqrt{x^2+5}}{3+\sqrt{x^2+5}} }[/tex]

[tex]\displaystyle{f(x)=\frac{(4-x^2)(3-\sqrt{x^2+5})}{9-x^2-5} }[/tex]

[tex]\displaystyle{f(x)=\frac{\cancel{(4-x^2)}(3-\sqrt{x^2+5})}{\cancel{(4-x^2)}} }[/tex]

[tex]\displaystyle{f(x)=3+\sqrt{x^2+5} }[/tex]

Fungsi bentuk akar kontinu untuk :

[tex]x^2+5\geq 0[/tex]

Karena x² + 5 merupakan definit positif (nilainya selalu ≥ 0), maka f(x) akan kontinu di semua titik.

.

KESIMPULAN

Fungsi [tex]\displaystyle{f(x)=\left ( \frac{3-\sqrt{x^2+5}}{4-x^2} \right )^{-1}}[/tex] tidak memiliki titik diskontinu atau fungsi kontinu di semua titik.

.

.

lim x→∞ (2x - 1) - √(4x² + 7x - 12) = -11/4

Pembahasan

lim x→∞ √(ax² + bx + c) - √(ax² + qx + r)

= (b - q)/(2√a)

lim x→∞ √(2x² - 6x + 5) - √(2x² - 3x + 4)

= (-6 - (-3)) / (2√2)

= -3/(2√2)

= -3/4 √2

= lim x→∞ √(2x - 1)² - √(4x² + 7x - 12)

= lim x→∞ √(4x² - 4x + 1) - √(4x² + 7x - 12)

= (-4 - 7) / (2√4)

= -11/4
