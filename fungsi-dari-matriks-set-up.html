5. Jelaskan fungsi dari page,margin,header/footer,dan sheet pada page set up


- Page dapat difungsikan untuk mengatur orientasi kertas (anda bisa mensetting kertas secara portrait/ vertikal maupun landscape/ horizontal)

- Margin dapat digungsikan untuk mengatur jarak karakter di dokumen dengan batas kertas (batas atas, batas bawah, batas kanan dan batas kiri)

- Header/ footer dapat difungsikan untuk menambahkan catatan kepala atau catatan kaki pada dokumen (misalnya memberi nomor halaman di dokumen)

- sheet dapat difungsikan untuk menentukan set print area (fasilitas yang mengijinkan kita untuk mencetak bagian yang telah disetting)
