Jelaskan apa yang dimaksud dari Request dan Command!

As a noun: A request means an act of I = Knowledge and fame were in as great request as wealth among us now. (Sir W. Temple)

As a noun: A command means An order to do something = I was given a command to cease shooting.

As a verb: Request means to ask somebody to do something.= She requested permission to film at the Palace.

Sebagai kata kerja: Request berarti meminta seseorang untuk melakukan sesuatu. = Dia meminta izin untuk syuting di Istana.

5. jelaskan maksut dari perintah command prompt berikut ini - shutdown. -s. - shutdown - R​


Perintah Shutdown Ubuntu dan CentOS

Perintah ‘shutdown’ Ubuntu dan CentOS ini berfungsi untuk mematikan, reboot, dan menghentikan sistem perangkat. Di bawah ini adalah perintah syntax untuk shutdown dan restart.

Shutdown [OPTIONS] [TIME] [WALL]

Argumen pertama dari [OPTIONS] adalah string waktu. Setelah memilih argumen ini, selanjutnya Anda dapat mengetikkan wall message untuk memperingati user yang sudah login bahwa sistem akan dimatikan.

Format untuk string waktunya adalah hh:mm (jam/menit), sedangkan format waktunya sendiri menggunakan aturan 24 jam. Fungsi dari string ini untuk menunjukkan waktu kapan perintah shutdown akan dijalankan. Anda juga bisa menggunakan elemen +m yang mengindikasikan menit. Instruksi ini akan memerintahkan perangkat untuk mati secara otomatis dalam hitungan menit yang dikehendaki.

Untuk mematikan VPS saat itu juga, Anda bisa menggunakan syntax now yang adalah alias untuk +0. Jika Anda tidak menentukan argumen [time], maka Linux akan mengaturnya ke +1 secara default sebelum shutdown dilakukan.

Perlu diingat bahwa Anda harus menentukan argumen waktu jika hendak membuat wall message. File /run/nonlogin akan dibuat 5 menit sebelum sistem dinonaktifkan asalkan Anda telah menetapkan argumen waktu untuk memperingati user yang hendak login.

13. perintah perintah command prompt (cmd) pada windows?​


Jawaban:

-. append = Perintah append memungkinkan Anda untuk membuka file yang ada dalam direktori lain

append c:\docs;c:\letters edit myfile.txt

-. arp = perintah arp digunakan untuk mengelola ARP atau address resolution protocol.

arp -s 220.0.0.161 00-50-04-62-F7-23

-. assoc =

Perintah assoc digunakan untuk memeriksa ekstensi file yang diasosiasikan dengan sebuah jenis file. (assoc .txt)

-. at =Perintah at digunakan untuk mengotomatisasi berbagai perintah lainnya pada waktu yang ditentukan. (at 1 /delete)

Jika ingin menghapus semuanya, Anda dapat mengetikkan perintah berikut:(at /delete /yes)

-. attrib =menemukan file yang tidak diinginkan, tetapi tidak dapat Anda hapus? Barangkali atribut file tersebut di-set ke read-only. (attrib -R example.txt)

-. bcdedit =bcdedit adalah perintah yang memungkinkan Anda untuk melakukan tiga hal, yaitu mengelola penyimpanan data boot configuration, menyimpan parameter konfigurasi tersebut, dan mengatur booting sistem operasi. (bcdedit /command <argument>)

-. bootsect =Perintah bootsect digunakan untuk memperbarui master boot code dalam partisi hard drive. (bootsect /nt52 E:)

-. break =Tergantung dengan pengaturannya, kombinasi tombol ctrl + C dapat digunakan untuk menghentikan sebuah proses MS-DOS. (break off)

-. change logon =Perintah change logon mengatur status logon di client session. (change logon /enable or change logon /disable or change logon /query)

-. chdir (chdir\windows\system32)

-. chkdsk (Perintah ini berfungsi untuk memeriksa status hard drive Anda. (chkdsk d: /r)

Command Prompt atau DOS prompt adalah sebuah command line (baris perintah) pada sebuah OS (Operating System) berbasis GUI untuk mengesekusi file dengan cara menuliskan perintahnya pada jendela cmd. 40. Jelaskan lima perintah command prompt yg anda kerahui


Jawaban:

1. Ping

Perintah ping ini adalah yang paling sering digunakan. Biasanya mereka menggunakannya untuk mengetahui konektivitas pada jaringan. Perintah ping ini mengirim ICMP echo request packets ke tujuan.

2. Tracert

Pada dasarnya perintah tracert ini hampir mirip dengan perintah ping, yang membedakannya adalah pada perintah tracert ini kita akan diperlihatkan jalur mana saja yang dilewati oleh packet yang dikirim.

3. Ipconfig

Perintah ipconfig bisa digunakan untuk mengkonfigurasi interface jaringan atau sekedar melihat informasi interface jaringan yang terdapat pada komputer Anda.

4. Nslookup

Perintah dasar lainnya adalah nslookup. Perintah ini digunakan untuk mencari alamat IP yang dipakai pada suatu domain.

5. Netstat

Netstat atau statistik jaringan. Perintah ini akan menampilkan koneksi jaringan yang masuk dan keluar serta informasi jaringan lain.
