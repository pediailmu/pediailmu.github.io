Penjelasan dengan langkah-langkah:

Diberikan:

rumus fungsi f(2x + 4) = 6x + 8.f(p) = 8f(3) = q

Kita akan menentukan nilai p + qp + q.

CARA PERTAMA

f(p) = 8

Parameter fungsi:
p = 2x + 4Nilai fungsi:
8 = 6x + 8
⇒ 6x = 0
⇒ 2x = 0Karena 2x = 0, maka:
p = 0 + 4
⇒ p = 4

f(3) = q

Parameter fungsi:
3 = 2x + 4
⇒ 2x = 3 – 4 = –1
⇒ x = –½Nilai fungsi:
q = 6x + 8
⇒ q = 6·(–½) + 8
⇒ q = –3 + 8
⇒ q = 5

Maka:
p + qp + q = 4 + 5·4 + 5
⇒ p + qp + q = 4 + 20 + 5
⇒ p + qp + q = 29
_______________

CARA KEDUA

f(2x + 4) = 6x + 8
⇒ f(2x + 4) = 6x + 12 – 4
⇒ f(2x + 4) = 3(2x + 4) – 4
Substitusi 2x + 4 menjadi x.
⇒ f(x) = 3x – 4

f(p) = 8
⇒ 3p – 4 = 8
⇒ 3p = 8 + 4 = 12
⇒ p = 12/3
⇒ p = 4f(3) = q
⇒ 3·3 – 4 = q
⇒ 9 – 4 = q
⇒ q = 5

Kita mendapatkan nilai yang sama dengan yang didapatkan pada cara pertama di atas, yaitu:
p + qp + q = 29

Berapakah koefisien elastisitas permintaan karena perubahan harga untuk teh botol ? b. Berapakah elastisitas permintaan karena perubahan pendapatan terhadap teh botol ? c. Berapakah elastisitas silang antara the botol dengan barang-barang lain? d. Berapakah jumlah teh botol yang diminta, akan naik atau turun bila terjadi perubahan bersama-sama pada pendapatan riel, harga teh botol dan harga barang-barang lain, masing-masing dengan 10 persen ?​


Jawaban:

ln Q = 177,6 + 0,023 lnY - 1,040 lnP + 0,939 lnR

a. Elastisitas harga = Δ%Q/Δ%P = d lnQ / d lnP

Elastisitas = A = -1,040 (Elastis)

b. Elastisitas pendapatan = Δ%Q/Δ%Y = d lnQ / d lnY

Elastisitas = B = 0,023 (Barang Normal)

c. Elastisitas Silang = Δ%Q / Δ%R = d lnQ / d lnR

Elastisitas = C = 0,939

d.Total Differensial:

ΔQ% = A . ΔP% + B . ΔY% + C . ΔR%

ΔQ% = -1,040 (10%) + 0,023 (10%) + 0,939 (10%)

ΔQ% = (-1,040 + 0,023 + 0,939) (10%)

ΔQ% = -0,78%

Saat terjadi peningkatan pendapatan riil, harga teh botol, dan harga barang-barang lain masing-masing 10%, maka jumlah permintaan akan berkurang sebesar 0,78%

10. 10. Harga sepatu di pasar mengalami peningkatan sebagai akibat banyaknya permintaan terhadap sepatu pada awal tahun pelajaran. Sepatu yang biasanya seharga Rp150.000,00 perpasang, naik menjadi Rp175.000,00. Jenis elastisitas penawaran sepatu jika fungsi penawarannya Q = 0,2P + 1000 adalah .... A. unitary C. in-elastis D. elastis sempurna E. in elastis sempurna B. elastis​


Jawaban:

Untuk menentukan jenis elastisitas penawaran sepatu, kita perlu memperhatikan persamaan fungsi penawaran yang diberikan (Q = 0,2P + 1000) dan perubahan persentase dalam harga (peningkatan harga sebesar Rp25.000,00 dari Rp150.000,00 menjadi Rp175.000,00).

Elastisitas penawaran dapat dihitung dengan rumus:

Elastisitas Penawaran (Ep) = (% perubahan jumlah penawaran) / (% perubahan harga)

Kita perlu menghitung perubahan jumlah penawaran dan perubahan harga dalam bentuk persentase:

Perubahan Jumlah Penawaran = (Q baru - Q lama) / Q lama

= ((0,2P + 1000) - (0,2P + 1000 - 25)) / (0,2P + 1000)

= 25 / (0,2P + 1000)

Perubahan Harga = (Harga baru - Harga lama) / Harga lama

= (175.000 - 150.000) / 150.000

= 25.000 / 150.000

Kemudian kita substitusikan nilai perubahan jumlah penawaran dan perubahan harga ke dalam rumus elastisitas penawaran:

Ep = (25 / (0,2P + 1000)) / (25.000 / 150.000)

= (150.000 * 25) / (0,2P + 1000) / 25.000

Simplifikasi:

Ep = 150.000 / (0,2P + 1000)

Dari nilai elastisitas penawaran yang didapatkan, kita dapat menentukan jenis elastisitasnya. Jika nilai Ep > 1, maka penawaran bersifat elastis. Jika nilai Ep = 1, maka penawaran bersifat unitary. Jika nilai Ep < 1, maka penawaran bersifat in-elastis.

Karena tidak diberikan nilai spesifik untuk P (harga), kita tidak dapat secara pasti menentukan jenis elastisitasnya. Namun, berdasarkan persamaan elastisitas penawaran yang diperoleh (Ep = 150.000 / (0,2P + 1000)), kita dapat menyimpulkan bahwa jawaban yang paling sesuai adalah B. elastis, karena elastisitas penawaran dapat melebihi nilai 1 dengan kenaikan harga yang relatif kecil.
