9. diketahui fungsi fx = 7 x min 8 Tentukan nilai x jika fx = 20​


Jawaban:

Untuk mencari nilai \(x\) ketika \(f(x) = 20\) pada fungsi \(f(x) = 7 \min 8\), kita harus memahami bahwa "min" mengacu pada nilai minimum antara \(7\) dan \(8\). Karena \(7\) lebih kecil daripada \(8\), maka nilai minimumnya adalah \(7\). Oleh karena itu, kita bisa menuliskan:

\(f(x) = 7 \min 8 = 7\)

Jadi, \(f(x) = 7\) untuk semua nilai \(x\). Artinya, tidak ada nilai \(x\) tertentu yang akan membuat \(f(x) = 20\) dalam fungsi ini. Jadi, jawaban akhirnya adalah bahwa tidak ada nilai \(x\) yang memenuhi \(f(x) = 20\) dalam fungsi ini.


10. tentukanlah titik balik fungsi fx = x⁴ - x²


Keadaan stasioner f'(x)=0
Maka 4x^3 - 2x = 0
Dan sederhanakan utk pemfaktoran dgn bagi dua, menjadi 2x^3 - x = 0.
Faktorkan x(2x^2 - 1)= 0
Diperoleh akar2 yaitu x = 0, x = - (akar2)/2 dan x = (akar 2)/2.
Masukkan tiap2 akar x tsb ke dlm f(x) utk mendapatkan nilai min/max.
Hasil uji tanda dgn memasukkan x ke f'(x) lalu dicek tanda+/-

.......(-)... -akar2 / 2....(+).... 0 ....(-)..... akar2 / 2. ....(+)........

Hasil2nya adalah (dgn posisi x berturut-turut dari kecil ke besar):
f(-akar2 / 2) = -1/4, f(0) = 0, dan f(akar2 / 2) = -1/4.

Jadi titik-titik balik minimum adalah (-akar2 / 2, -1/4) dan (akar2 / 2, -1/4) sedangkan titkk balik maksimum adalah (0,0)
