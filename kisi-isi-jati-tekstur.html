32. jelaskan cara pengisian warna tekstur pada sebuah objek​


Jawaban:

1. Smart Fill Tool

Tool Ini terletak di bagian tengah Toolbox, Smart Fill Tool berfungsi untuk memberi warna atau menghilangkan warna pada objek atau bagian dari objek.

2. Fill Tool

Fill Tool terletak di posisi ke-2 dari bawah Toolbox, berfungsi untuk memberi warna sesuai dengan mode yang dipilih :

Uniform Fill : Untuk mewarnai dengan satu warna (Polos)

Fountain Fill : Untuk mewarnai dengan Gradasi warna

Pattern Fill : Untuk mewarnai dengan Pattern

Texture Fill : Untuk mewarnai dengan tekstur

3. Interactive Fill Tool

Interactive Fill Tool berada di paling bawah Toolbox, fungsinya hampir sama dengan Fountain Fill hanya saja Interactive Fill Tool lebih mudah dalam mengatur arah gradasi warnanya.

4. Color Pallete

Color Pallete berada di bilah sisi kanan layar, digunakan untuk mewarnai objek yang terseleksi (objek yang akan diwarnai diseleksi dulu) klik warna pada Color Pallete untuk mewarnai Objek.

5. Fill Color

Fill Color berada di pojok kiri-bawah layar, digunakan untuk mewarnai objek yang terseleksi, sama seperti Color Pallete bedanya di Fill Color terdapat semua pilihan warna.

6. Outline Color

Berada tepat di bawah Fill Color, digunakan untuk mewarnai garis luar pada Objek.

37. tolong bantuin isi kisi kisi dong ka​


1. Start Pendek (Short Start/Bunch Start)

Posisi kedua kaki pada posisi ini bisa dibilang sangat dekat dan sulit untuk dilakukan kecuali oleh para atlet yag terlatih.

Kaki terbuka selebar bahu, kaki depan menginjak papan start depan, kaki belakang menginjak papan start belakang.

Lutut kaki belakang berada pada posisi sejajar dengan jari-jari kaki depan atau bahkan lebih depan kurang lebih satu jengkal dari telapak kaki depan.

Jika teknik ini berhasil dilakukan, tolakan yang dihasilkan dari posisi ini akan menghasilkan lari dengan kecepatan tinggi.

2. Start Medium

Posisi ini sering dipergunakan oleh para pelari baik pelari profesional ataupun pelari amatir.

Berbeda dengan start pendek, pada start medium ini letak lutut kaki belakang berada sejajar dengan tungkai kaki depan atau sedikit lebih kebelakang dari kaki depan. Posisi ini juga masih bisa menghasilkan kecepatan yang tinggi.

3. Start Panjang

Posisi ini jarang sekali dipakai kecuali oleh para pelari yang memiliki kaki yang panjang. Posisi lutut kaki belakang berada sekitar 30 cm dari kaki depan atau menyesuaikan. Yang jelas, posisi lutut kaki belakang selalu berada dibelakang kaki depan.
