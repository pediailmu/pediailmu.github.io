Inline Style adalah bentuk pengaplikasian CSS ke HTML pada sebuah tag html dengan menggunakan atribut style.

- Embedded Style : pengaplikasiaan CSS ke html lewat tag <style> di dalam tag <head>

- Linked Style :  pengaplikasiaan CSS ke html lewat tag <link> di dalam tag <head>, pada style ini aturan CSS berada di file yang berbeda dengan file html, dengan format file yaitu .css
