14. ikon apa saja yang ada open office.org dan fungsinya


Beberapa icon di open office.org antara lain:
- New: berfungsi untuk membuat dokumen baru.
- Open: berfungsi untuk membuka dokumen yang sebelumnya telah dibuat.
- Save: berfungsi untuk menyimpan dokumen.
- Save As PDF: berfungsi untuk menyimpan dokumen dengan format PDF.
- Print: berfungsi untuk mencetak dokumen.
- Cut: berfungsi untuk memotong data yang diseleksi.
- Copy: berfungsi untuk menyalin data yang diseleksi.
- Paste: berfungsi untuk menempel data yang sebelumnya di-cut atau di-copy.

31. Apa fungsi: Fungsi open? Fungsi prepare? Fungsi send? Fungsi publish?


Jawaban:

1. Open berfungsi untuk membuka dokumen yang sebelumnya pernah disimpan.

2. Prepare berfungsi untuk mempersiapkan file supaya siap didistribusikan.

3. Send berfungsi untuk mengirim dokumen via e-mail dan internet fax.

4. Publish berfungsi untuk mendistribusikan dokumen Ms. Word ke media blog.

Penjelasan:

Open, prepare, Send dan Publish ialah contoh-contoh submenu dalam Office button pada program Ms. Word 2007. Perhatikan gambar berikut ini (sesuaikan nomornya).
