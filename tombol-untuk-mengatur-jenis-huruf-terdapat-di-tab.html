38. tombol tab pada keyboard unyuk mengatur​


Jawaban:

1. Mengatur bagian awal paragraf

Adapun fungsi tombol tab yang pertama pada sebuah keyboard baik komputer atau laptop yakni untuk mengatur bagian awal paragraf. Jika kalian ingin membuat baris pertama sebuah paragraf bisa menjorok ke dalam, maka kalian hanya perlu memencet satu tombol saja, yakni tombol tab. Jika kalian menekannya maka secara otomatis untuk baris pertama akan bergeser ke dalam. Dengan tab kalian tak perlu lagi menggunakan tombol spasi secara berulang-ulang untuk mengatur baris pertama tersebut lebih menjorok ke dalam.

2. Mempermudah proses login

Adapun fungsi tombol tab kedua yakni untuk mempermudah proses login. Jika kalian ingin login ke Facebook, email, atau sejumlah situs yang lainnya, untuk mempermudah kalian berpindah dari isian satu ke isian lainnya, maka kalian dapat memakai tombol. Kalian cukup menekan tombol Tab, lalu secara otomatis kursor berpindah menuju kolom lainnya. Dalam hal ini kalian tidak perlu repot menggunakan Mouse untuk pindah ke kolom username atau kolom password.

3. Mempermudah proses pengisian data online

Untuk fungsi tombol tab pada keyboard selanjutnya yakni untuk mempermudah kalian dalam pengisian data online. Ketika kalian ingin mengisi data online, tentu kalian perlu mengisi banyak kolom. Untuk perpindahan kolom dengan mudah kalian cukup menekan tombol Tab aja.

Jawaban:

Untuk mengatur bagian awal pada alinea/paragraf agar menjorok ke dalam.
