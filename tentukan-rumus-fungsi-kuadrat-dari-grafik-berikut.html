2. tentukan rumus fungsi kuadrat dari grafik berikut


Penjelasan dengan langkah-langkah:

Diket :

• Titik puncak (2, 0); xp = 2 dan yp = 0

• memotong sumbu y di titik (0, -4}

x = 0 dan y = -4

Tentukan fungsi kuadratnya !

Jawab :

y - yp = a(x - xp)²

-4 - 0 = a(0 - 2)²

-4 = a(-2)²

-4 = 4a

a = -1

Maka, fungsi kuadratnya adalah

y = a(x - xp)² + yp

y = - (x - 2)² + 0

y = - (x² - 4x + 4)

y = -x² + 4x - 4
