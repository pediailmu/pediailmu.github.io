16. Tuliskan sebuah program (codingan) yang bisa menampilkan fungsi, push, pop, clear, cetak, dan exit Dalam sebuah program stack!


krisanalfa / pop-push.cpp

[C++] POP PUSH dalam C++ menggunakan satu array dan dua stack.
pop-push.cpp
/**
|----------------------------------------------------------------------------------
| README FIRST
|----------------------------------------------------------------------------------
| I HATE WINDOWS, SO I USE LINUX TO MAKE THIS SHIT WORKS
| BUT I LOVE KNOWLEDGE, SO I MAKE THIS ONE NOW WORKS FOR WINDOWS TO
| I HATE C++
| BUT I LOVE ANGELA CRISANTI, SO I DO THIS SHIT
| COMPILE WITH: g++ pop-push.cpp -o shitty
| RUN WITH : ./shitty
*/

#include
#include

#define MAX 50
#define true 1
#define false 0

char stack[MAX];
int top1, top2;

// Inisialisasi stack
void init() {
top1 = 0;
top2 = MAX+1;
}

// Mencari tau apakah stack masih bisa digunakan apa enggak
int full(void) {
if(top1 + 1 >= top2) {
return true;
} else {
return false;
}
}

// Mencari tau apakah stack 1 atau stack 2 masih kosong apa enggak
int empty(int nomorstack) {
switch(nomorstack) {
case 1:
if(top1 == 0) {
return true;
} else {
return false;
}
break;
case 2:
if(top2 == MAX + 1) {
return true;
} else {
return false;
}
break;
default: printf("Nomor stack salah");
break;
}
}

// Buat hapus semua data stack di dalam array
void clear(int nomorstack) {
switch(nomorstack) {
case 1:
top1 = 0;
break;
case 2:
top2 = MAX+1;
break;
default: printf("Nomor stack salah");
break;
}
}

// Buat push data ke dalam stack array, misalnya ---> push('sesuatu', 1);
// Ini berarti kita push string 'sesuatu' ke stack pertama
void push(char data, int nomorstack) {
if(!full()) {
switch(nomorstack) {
case 1:
top1++;
stack[top1] = data;
break;
case 2:
top2--;
stack[top2] = data;
break;
default: printf("\nNomor stack salah");
break;
}
} else {
printf("\nStack penuh");
}

}

// Buat pop data stack misal mau pop stack pertama ---> pop(1);
// Kalau mau dicetak tinggal gini ---> std::cout << pop(1);
char pop(int nomorstack) {
char data;
if(!empty(nomorstack)) {
switch(nomorstack) {
case 1:
data = stack[top1];
top1--;
return data;
break;
case 2:
data = stack[top2];
top2++;
return data;
break;
default: printf("\nNomor stack salah");
break;
}
} else {
printf("\nStack masih kosong");
}

return 0;
}

// Tampilkan semua data stack di dalam array
void display() {
if ((top1 = 0) && (top2 = MAX + 1)) { std::cout << "(empty)\n"; return; }
for (int t = 0 ; t < MAX ; t++) std::cout << stack[t] << " ";
std::cout << "\n";
}

// Program berjalan
int main() {
// Inisialisasi stack, yang ini harus jalan dulu
init();

// Masukin 'a' ke dalam stack pertama
push('a', 1);
// Masukin 'n' ke dalam stack pertama
push('n', 1);
// Masukin 'u' ke dalam stack pertama
push('u', 1);

// cetak pop dari stack pertama
std::cout << "Hasil pop stack pertama: " << pop(1) << "\n";

// cetak array termasuk kedua stack
display();

std::cout << "\n";

// Masukin '*' ke dalam stack kedua, char '*' gak akan masuk karena udah ke pop
push('*', 2);
// Masukin 's' ke dalam stack kedua
push('s', 2);
// Masukin 'e' ke dalam stack kedua
push('e', 2);
// Masukin 'm' ke dalam stack kedua
push('m', 2);
// Masukin 'e' ke dalam stack kedua
push('e', 2);
// Masukin 'g' ke dalam stack kedua
push('g', 2);

// cetak pop dari stack kedua
std::cout << "Hasil pop stack kedua: " << pop(2) << "\n";
// cetak array termasuk kedua stack
display();

// exit with no error
// http://en.wikipedia.org/wiki/Error_code
return 0;
}

21. apakah perbedaan antara clear safety glasses dan clear safety goggles jawabanpenjelasan ​


Jawaban:

Perbedaan antara clear safety glasses (kacamata keselamatan transparan) dan clear safety goggles (kacamata pelindung transparan) adalah:

1. Desain: Clear safety glasses umumnya memiliki desain yang mirip dengan kacamata biasa. Mereka terdiri dari dua lensa transparan yang dihubungkan dengan jembatan hidung dan dua penyangga di belakang telinga. Sedangkan clear safety goggles, memiliki desain yang menutupi lebih banyak area wajah. Mereka umumnya memiliki lensa transparan yang luas, penyangga hidung dan tali kepala yang dapat disesuaikan.

2. Perlindungan: Clear safety glasses dan clear safety goggles, keduanya memberikan perlindungan terhadap partikel, cipratan bahan kimia dan debu yang ada di lingkungan kerja. Namun, clear safety goggles biasanya memberikan perlindungan yang lebih baik karena desainnya yang menutupi lebih banyak area wajah, termasuk samping dan bawah mata. Hal ini membantu mencegah partikel atau bahan kimia masuk melalui celah di sisi kacamata.

3. Kegunaan: Clear safety glasses umumnya digunakan di lingkungan kerja di mana risiko partikel kecil atau debu terjadi, misalnya dalam industri manufaktur, konstruksi, atau laboratorium. Mereka juga sering digunakan di sektor medis atau rumah sakit. Sementara clear safety goggles lebih sering digunakan di lingkungan di mana risiko percikan bahan kimia, cairan berbahaya, atau partikel yang lebih besar terjadi, seperti di industri kimia, percetakan, atau saat bekerja dengan bahan berbahaya.


22. Tuliskan perbedaan clear dan Clear all dalam pembuatan tabulasi​


Jawaban:

Penjelasan:

Clear: Saat menggunakan perintah "clear" pada tabulasi, itu berarti menghapus atau mengatur ulang tabulasi pada posisi yang sedang aktif atau dipilih. Jadi, jika Anda menggunakan "clear" pada tabulasi tertentu, itu hanya akan menghapus atau mengatur ulang tabulasi pada posisi tersebut tanpa memengaruhi tabulasi lainnya dalam dokumen.

Clear All: Saat menggunakan perintah "Clear All" pada tabulasi, itu berarti menghapus semua tabulasi yang ada dalam dokumen. Ini akan menghapus semua tabulasi yang telah Anda atur pada semua paragraf atau teks di dokumen, termasuk tabulasi kiri, tengah, kanan, desimal, dan tabulasi khusus lainnya. Dengan menggunakan "Clear All", Anda dapat menghapus semua tabulasi sekaligus tanpa perlu menghapusnya satu per satu.

Dengan kata lain, "clear" digunakan untuk menghapus atau mengatur ulang tabulasi pada posisi yang sedang aktif atau dipilih, sedangkan "Clear All" digunakan untuk menghapus semua tabulasi dalam dokumen.
