5. sebutkan fungsi tombol-tombol pada komputer​


Tombol keyboard:

Caps Lock = untuk membesarkan (mengkapitalkan) semua hurufWindows = untuk ke main source (untuk shutdown, pencaharian, dll)Shift = untuk membesarkan satu huruf dlm sebuah kalimatAlt = untuk commandCtrl = untuk command (contoh Ctrl + P untuk mengeprint, ctrl + v untuk paste, ctrl + c untuk copy)F1 - F12 = pengaturan komputer luarNum Lock = tombol yang memiliki fungsi ganda (contoh: Num Lock on, fungsi pengetikan angka-angka dan operator matematik aktif. Num Lock off, fungsi tombol navigasi aktif.)ESC (Escape): adalah tombol yang kegunaanya sama dengan pilihan CANCEL, yaitu untuk membatalkan satu tahap pekerjaan.Enter: adalah tombol yang kegunaanya sama dengan pilihan tombol OK, yaitu untuk menyatakan bahwa operasi yang dilakukan betul dan selesai. Enter juga dapat berarti menyisipkan baris kosong atau baris baru pada proses pemasukan text. TAB : Berarti pindah ke field atau daerah isian atau pilihan berikutnya. Pada saat proses pengetikan, TAB berarti lompat ke penghentian TAB (tab-stop) terdekat.

Backspace: akan mengakibatkan karakter (huruf) disebelah kiri kursor terhapus.

 Delete: akan mengakibatkan karakter(huruf) disebelah kanan atau ditempat dimana kursor berada akan terhapus.

 Home: akan mengakibatkan kursor berpindah ke awal baris dimana kursor berada.

 End: akan mengakibatkan kursor berpindah ke akhir baris dimana kursor berada.

Page Up: akan mengakibatkan kursor berpindah ke atas sejauh satu layar dari kursor berada.

Page Down: akan mengakibatkan kursor berpindah ke bawah sejauh satu layar dari kursor berada. Up    : Memindahkan kursor 1 baris ke atasDown : Memindahkan kursor 1 baris ke bawah Left : Memindahkan kursor 1 karakter ke kiriRight : Memindahkan kursor 1 karakter ke kanan

25. Sebutkan 4 tombol keyboard yang dapat kamu gunakan untuk memerintah komputer. Sebutkan pula perintah yang dapat kamu berikan menggunakan tombol-tombol tersebut.


Jawaban:

Misalkan komputer memakai sistem operasi Windows 7:

1. Tombol F1

2. Tombol F3

3. Tombol F5

4. Tombol logo Windows

Penjelasan:

Dalam tampilan desktop (tidak membuka aplikasi apapun), masing-masing tombol di atas memiliki perintah tersendiri:

1. Tombol F1 yaitu tombol perintah untuk menampilkan Windows Help and Support

2. Tombol F3 yaitu tombol perintah untuk menampilkan Search File (untuk mencari file di komputer)

3. Tombol F5 yaitu tombol perintah untuk melakukan refresh tampilan desktop

4. Tombol logo Windows yaitu tombol untuk menampilkan "menu mulai" atau Start menu.

Untuk lebih jelasnya, perhatikan keyboard berikut (di bagian yg dilingkari kuning).
