1. petanyaan matematika


Jawab: Sebuah kapal laut yang berkapasitas tidak lebih dari 500 penumpang memiliki kelas utama dan kelas ekonomi dengan kelas utama diperbolehkan membawa bagasi paling banyak 80 kg dan kelas ekonomi 20 kg. Jika kapal laut hanya mampu membawa bagasi total sebanyak 16.000 kg. Tarif kelas utama Rp 250.000,- dan kelas ekonomi Rp 125.000,-. Pendapatan maksimum yang akan diterima oleh angkutan kapal laut tersebut adalah Rp 75.000.000,-


Penjelasan dengan langkah-langkah:

Soal diatas diselesaikan dengan persamaan linear dua variabel. Pertama kita tuliskan terlebih dahulu persamaannya. Kelas utama dituliskan sebagai variabel x dan kelas ekonomi sebagai variabel y. Pada soal diketahui kapasitasnya tidak lebih dari 500 penumpang untuk kelas utama maupun ekonomi, berarti persamaannya:

x + y = 500

Penumpang kelas utama diperbolehkan membawa bagasi paling banyak 80 kg dan kelas ekonomi 20 kg dan kapal laut hanya mampu membawa bagasi total sebanyak 16.000 kg, persamaannya adalah:

80x + 20y = 16.000 disederhanakan menjadi 4x + y = 800

Eliminasi variabel y untuk mengetahui nilai x

4x + y = 800

x + y = 500      -

3x = 300

x = 100

Substitusikan nilai x ke persamaan x + y = 500 untuk mendapatkan nilai y

100 + y = 500

y = 500 - 100

y = 400

Penumpang kelas utama adalah 100 orang dan kelas ekonomi adalah 400 orang. Pendapatan maksimum dari angkutan kapal laut tersebut adalah:

pendapatan maksimum = 100 ( 250.000 ) + 400 ( 125.000 )

pendapatan maksimum = 25.000.000 + 50.000.000

pendapatan maksimum = 75.000.000

Jadi pendapatan maksimum yang bisa diperoleh angkutan kapal laut tersebut adalah Rp 75.000.000,-

16. PERTANYAAN MATEMATIKA ‼️ ​


Rumus untuk luas permukaan prisma adalah:

Luas Permukaan = 2 × (Luas alas + Luas sisi

Untuk menghitung luas belah ketupat, kita dapat menggunakan rumus:

Luas Belah Ketupat = 0.5 × d1 × d2

Di mana d1 dan d2 adalah panjang diagonal belah ketupat. Dalam kasus ini, d1 = 18 cm dan d2 = 24 cm. Jadi,

Luas Belah Ketupat = 0.5 × 18 cm × 24 cm = 216 cm²

Kemudian, kita dapat menggantikan nilai ini ke dalam rumus luas permukaan prisma:

1692 cm² = 2 × (216 cm² + Luas sisi)

Sekarang, kita bisa mencari nilai Luas sisi:

Luas sisi = (1692 cm² - 2 × 216 cm²) / 2 = (1692 cm² - 432 cm²) / 2 = 1260 cm² / 2 = 630 cm²

Dalam prisma berbentuk belah ketupat, luas sisi adalah luas salah satu sisi yang identik. Luas sisi tersebut dapat dihitung dengan rumus:

Luas sisi = panjang sisi × tinggi

Kita ingin mencari tinggi prisma, jadi kita dapat menyusun rumus sebagai berikut:

tinggi = Luas sisi / panjang sisi

Dalam hal ini, kita tahu luas sisi (630 cm²), dan panjang diagonal (d1) belah ketupat adalah 18 cm. Jadi,

tinggi = 630 cm² / 18 cm = 35 cm

Jadi, tinggi prisma tersebut adalah 35 cm.

Penjelasan dengan langkah-langkah:

a² + b² = c²

9² + b² = c²

81 + b² = c²

b² = c² - 81

a² + b² = c².

12² + b² = c²

144 + b² = c²

b² = c² - 144

1692 cm².

Luas Permukaan = 2((Diagonal 1 x Tinggi) + (Diagonal 2 x Tinggi) + (Panjang Alas x Lebar Alas))

1692 = 2((18 x c) + (24 x c) + (x x y))

1692 = 2(18c + 24c + xy)

1692 = 2(42c + xy)

846 = 42c + xy

846 = 42c + xy

846 = 42c + x²

b² = c² - 81

x² = 42c

846 = 42c + 42c

846 = 84c

c = 10.0714 cm

b² = 10.0714² - 81

b² = 101.4286

b = 10.0714 cm (tinggi prisma)

Jadi, tinggi prisma tersebut adalah 10.0714 cm atau dapat dinyatakan sebagai 11 cm dengan pembulatan ke atas.

27. •MATEMATIKA • Pertanyaan ada di gambar.​


PENYELESAIANSOAL 1NOMOR A

x² - 4 = 0

(x + 2)(x - 2) = 0

x + 2 = 0 Atau x - 2 = 0

x = -2               x = 2

Maka, akar-akar persamaan kuadrat dari x² - 4 = 0 adalah -2 dan 2

NOMOR B

x² - 625 = 0

(x + 25)(x - 25)

x + 25 = 0 Atau x - 25 = 0

x = -25               x = 25

Jadi,Akar - Akar persamaan dari x² - 625 = 0 adalah -25 dan x = 25

NOMOR C

x² - 1/16 = 0

(x + 1/3)(x - 1/2)

x + 1/3 = 0 Atau x = 1/2

x = -1/3

Maka,Akar - Akar persamaan kuadrat dari x² - 1/16 = 0 adalah -1/3 dan 1/2

SOAL 2NOMOR A

x² - 2x = 0

(x + 1)(x - 2)

x + 1 = 0 Atau x - 2 = 0

x = -1               x = 2

Jadi,Akar - akar persamaan kuadrat dari x² - 2x = 0 adalah x = -1 dan x = 2

NOMOR B

6x² + 3x = 0

(3x - 1)(2x + 3)

3x - 1 = 0 Atau 2x + 3 = 0

3x = 0 + 1         2x = 0 - 3

3x = 1               2x = -3

x = 1/3                x = -3/2

Maka,Akar - Akar Persamaan kuadrat dari 6x² + 3x = 0 adalah -3/2 dan 1/3

NOMOR C

3x² - 2x = -x

3x² - 2x + x = 0

(x - 1)(3x - 1)

x - 1 = 0 Atau 3x - 1 = 0

x = 1                3x = 0 + 1

                       3x = 1

                         x = 1/3

Maka,Akar - Akar persamaan kuadrat dari 3x² - 2x = -x adalah x = 1/3 dan 1

NOMOR D

2/x = 1/x²

2/x + 1/x²  

(x - 1)(2x - 1)

x - 1 = 0 Atau 2x - 1 = 0

x = 0 + 1         2x = 0 + 1

x = 1               2x = 1

                       x = 1/2

Maka,Akar - Akar persamaan kuadrat dari 2x = 1/x² adalah 1/2 dan 1
