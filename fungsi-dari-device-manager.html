7. jelaskan tentang Input device,Process device, Out put device beri masing² contoh beserta fungsinya​


Input device mengirimkan informasi ke sistem komputer untuk diproses, dan Output device mereproduksi atau menampilkan hasil pemrosesan tersebut. Input device hanya mengizinkan input data ke komputer dan Output device hanya menerima output data dari perangkat lain. Process device adalah semua perangkat di dalam komputer yang menafsirkan dan memanipulasi data yang masuk selama tahap ini.

Contoh Input device yaitu: Keyboard, mouse, microfon, dan webcam.

Contoh Output device yaitu: Monitor, proyektor, dan speaker.

Contoh Process device yaitu: CPU, memory and motherboard.

Pembahasan

Sebagian besar perangkat hanya merupakan Input device atau Output device, karena masing-masing peragkat hanya dapat menerima input data dari pengguna atau data output yang dihasilkan oleh komputer. Namun, beberapa perangkat dapat menerima input dan menampilkan output, dan disebut sebagai perangkat I/O (perangkat input/output).

Contohnya keyboard mengirimkan sinyal listrik, yang diterima sebagai input. Sinyal tersebut kemudian ditafsirkan oleh komputer dan ditampilkan atau dikeluarkan pada monitor sebagai teks atau gambar. Di bagian bawah gambar, komputer mengirim, atau mengeluarkan data ke printer. Kemudian, data tersebut dicetak ke selembar kertas, yang juga dianggap sebagai output.

Process device merupakan perangkat apa pun di dalam komputer yang menafsirkan dan memanipulasi data yang masuk selama tahap ini.

Process device adalah komponen yang bertanggung jawab untuk pemrosesan informasi di dalam sistem komputer. Ini termasuk perangkat seperti CPU, memori dan motherboard. Perangkat penyimpanan adalah komponen yang memungkinkan data disimpan dalam sistem komputer.

19. Jelaskan fungsi dan manfaat perangkat alat bantu Device Manager


Manfaat Device Manager :
– Melihat apakah hardware yang terpasang berjalan dengan baik
– Mengubah pengaturan konfigurasi hardware
– Identifikasi driver yang sedang dijalankan (aktif) dan mendapatkan informasi detailnya
– Menginstall update driver
– Menon-aktifkan, mengaktifkan dan meng-uninstall driver
– Mengembalikan driver ke keadaan sebelumnya (roll back)
– Mencetak ringkasan daftar driver yang terinstall di computer

Fungsi kegunaan Device Manager, antara lain:
Menunjukkan status suatu hardware
Menunjukkan informasi detil suatu hardware
Mengelola driver hardware
Disable & Enable hardware
Mengidentifikasi konflik antar hardware, dll.
Device Manager paling sering digunakan untuk pengelolaan driver suatu hardware. Misalnya instalasi driver, uninstal driver, update driver, rollback driver, dan bermacam masalah yang berkaitan dengan driver suatu hardware.Device Manager adalah sebuah applet Control Panel yang ada di sistem operasi Microsoft Windows. Yang memungkinkan bagi kita nih sebagai pengguna untuk melihat dan mengontrol perangkat keras yang ada dan terinstal pada komputer kita. Trus ketika salah satu hardware yang ada di komputer tidak bekerja/berfungsi, maka hardware tersebut akan disorot dan diperlihatkan kepada pengguna untuk menanganinya. Daftar hardware yang ada dalam komputer dapat kita urutkan berdasarkan berbagai kriteria, bisa berdasarkan type, berdasarkan koneksi, dll.

Beberapa hal yang bisa dilakukan untuk setiap perangkat:
Memasok driver perangkat, yakni untuk perangkat yang ada di komputer kita tapi belum terinstal drivernya, maka kita bisa memasok driver tersebut dengan mencarinya dari berbagai sumber.Mengaktifkan atau menonaktifkan perangkat.Mengirim pesan ke Windows untuk mengabaikan perangkat rusak.Juga melihat beberapa sifat teknis lainnya.
Device Manager mulai diperkenalkan pada masa Windows 95 dan kemudianmulai ditambahkan juga ke Windows 2000. Trus pastinya berkembang deh sampai sekarang.

34. tuliskan item item device manager​


Device Manager dalam komputer Windows, adalah perluasan dari Microsoft Management Console. Device Manager menampilkan seluruh hardware yang bisa di-inisialisasi (dikenali) oleh Windows. Tampilannya sudah ter-organisir (dikelompokkan) sedemikian rupa sehingga akan memudahkan pengelolaan setiap hardware yang ada.

Membuka Device Manager Windows

Ada beberapa cara untuk membuka Device Manager, dan berikut adalah cara yang (menurut saya) adalah paling mudah.Membuka Device Manager Windows XP :
Klik-kanan icon My Computer -> pilihProperties -> pilih tab Hardware -> klik tombol Device Manager. Lihat gambar-3.Selain itu bisa juga membuka Device Manager dari Control Panel Windows XP :Start -> Control Panel -> Performance and Maintenance -> System -> selanjutnya seperti langkah diatas. Atau,Tekan bersamaan tombol [Windows +Pause Break].Membuka Device Manager Windows 7 :
Klik Start -> pada Search Box ketik Device Manager -> dari list pilih Device Manager. Lihat Gambar-4.Selain itu bisa juga membuka Device Manager dari Control Panel Windows 7 : Start -> Hardware and Sound -> padaDevices and Printers pilih Device Manager.Pada Windows 7 dan Windows 8 tekan tombol [Windows + R] dan ketik [devmgmt.msc]  tanpa kurung lalu tekanEnter.

36. Sebutkan langkah-langkah untuk memnuka device manager dan sebutkan apa saja isi dari halaman device manager


nstal maka device manager akan memberitahukan kita dengan menampilkan peringatan tepat disamping icon hardware yang terpasang tersebut. Yang perlu dilakukan adalah menginstal driver. Apabila tidak ada peringatan dan tidak ada simbol hardware, langkah yang harus dilakukan adalah melakukan pengecekan kembali keterpasangan hardware tersebut. Kemungkinan hardware belum terpasang dengan sempurna.

Melalui device manager kita bisa meangkatifkan atau menonaktifkan hardware yang terpasang.

Meminta bantuan Microsoft untuk menanggulangi permasalahan hardware, dengan mengirimkan pesan error kepada mereka melalui jendela device manager.

Selain itu melalui device manager kita juga bisa melihat informasi lain yang berhubungan dengan suatu hardware.

Cara Cepat Membuka Device Manager pada Semua versi Windows

Buka Run (tekan Tombol Windows + R bersamaan) lalu ketikkan devmgmt.msc dan tekan tombol OK.

Jendela Device Manager akan langsung muncul.
