25. Apa fungsi dari Save Auto Recover ????


Auto Save: Fitur untuk menyimpan dokumen secara otomatis di latar belakang setiap 10 menit sekali
Auto Recovery: Fitur untuk mengembalikan dokumen yang belum sempat kesave gara-gara program not responding atau komputer mati dengan sendirinya (ketika program tiba-tiba tidak berfungsi)
Semoga membantu...Sekarang Microsoft Excel memiliki fitur AutoRecover terpasang yang telah diganti tambahan AutoSave yang ada di versi Excel yang lebih awal dari Microsoft Excel 2002. Fitur AutoRecover dapat menyimpan salinan semua berkas Excel terbuka pada interval pengguna yangtetap dapat ditentukan. Berkas dapat dipulihkan jika Excel dapat menutup secara tiba-tiba, misalnya, selama kegagalan daya.

40. Auto increament berfungsi untuk


Fungsi dari Auto Increament itu sendiri adalah membuat angka berurut dari mulai terkecil sampai terbesar dan tidak perlu diisi secara manual ketika memasukkan data ke table Atribut AUTO_INCREMENT digunakan untuk tipe data numerik (biasanya tipe data INT), dimana jika kita menetapkan sebuah kolom dengan atribut AUTO_INCREMENT, maka setiap kali kita menginputkan data, nilai pada kolom ini akan bertambah 1. Nilai pada kolom tersebut juga akan bertambah jika kita input dengan NULL atau nilai 0.
