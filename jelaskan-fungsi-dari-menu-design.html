Penjelasan:

Aplikasi apa yang di gunakan ?

Walaupun tidak pasti aplikasi apa yang di gunakan, tapi sy coba deskripsikan salah satu aplikasi yakni Ms. Office :)

Berikut adalah beberapa bagian yang terdapat pada menu File, Home, Insert, dan Design pada Microsoft Office:

Menu File:

1. New - Untuk membuat dokumen baru.

2. Open - Untuk membuka dokumen yang sudah ada.

3. Save - Untuk menyimpan dokumen yang sedang diedit.

4. Print - Untuk mencetak dokumen.

5. Export - Untuk mengekspor dokumen ke format lain.

Menu Home:

1. Clipboard - Untuk menyalin, memotong, dan melepaskan teks dan gambar.

2. Font - Untuk mengubah jenis huruf, ukuran, dan gaya tulisan.

3. Paragraph - Untuk mengatur paragraf, seperti jarak antarbaris, perataan, dan indentasi.

4. Styles - Untuk mengatur gaya dokumen, seperti judul, subjudul, dan gaya blok teks.

5. Editing - Untuk melakukan operasi pengeditan seperti pencarian, penggantian, dan pemformatan.

Menu Insert:

1. Table - Untuk membuat tabel di dalam dokumen.

2. Picture - Untuk memasukkan gambar ke dalam dokumen.

3. Shapes - Untuk menambahkan bentuk-bentuk dasar, seperti kotak dan lingkaran.

4. SmartArt - Untuk membuat diagram dan grafik.

5. Hyperlink - Untuk menambahkan tautan ke dokumen atau website lain.

Menu Design:

1. Page Setup - Untuk mengatur tata letak halaman, seperti ukuran kertas, orientasi, dan margin.

2. Themes - Untuk memilih tema atau pola warna untuk dokumen.

3. Colors - Untuk mengatur palet warna untuk dokumen.

4. Fonts - Untuk mengatur jenis huruf untuk dokumen.

5. Effects - Untuk menambahkan efek visual ke dalam dokumen seperti bayangan dan refleksi.

21. Carilah fungsi menu file, menu slide show, menu insert, menu home, menu design, menu review and view, menu animation


Jawaban:

Menu-file: Menu file biasanya berisi fungsi-fungsi terkait file seperti membuka, menyimpan, mencetak, dan menutup presentasi atau dokumen yang sedang dikerjakan. Selain itu, menu file juga bisa digunakan untuk mengatur preferensi atau pengaturan aplikasi atau program yang digunakan.Menu-slide show: Menu slide show berisi fungsi-fungsi terkait tampilan presentasi, seperti memulai tampilan slide secara otomatis, mengatur durasi tampilan slide, menambahkan efek tampilan, dan lain sebagainya.Menu-insert: Menu insert berisi fungsi-fungsi terkait penambahan elemen baru ke dalam dokumen atau presentasi. Contohnya seperti menambahkan gambar, tabel, grafik, video, hyperlink, dan lain sebagainya.Menu-home: Menu home biasanya berisi fungsi-fungsi yang sering digunakan saat membuat dokumen atau presentasi, seperti pengaturan format font, paragraf, warna, alinea, dan lain sebagainya.Menu-design: Menu design berisi fungsi-fungsi terkait pengaturan tampilan atau desain presentasi, seperti memilih tema atau template, mengubah latar belakang, menyesuaikan warna, dan lain sebagainya.Menu-review and view: Menu review and view biasanya berisi fungsi-fungsi terkait pengecekan dan revisi dokumen atau presentasi, seperti menambahkan komentar, memeriksa ejaan dan tata bahasa, serta melakukan revisi atau perbaikan.Menu-animation: Menu animation berisi fungsi-fungsi terkait penambahan efek atau animasi pada tampilan presentasi, seperti animasi masuk, animasi keluar, dan animasi pergerakan objek lainnya. Fungsi-fungsi ini membantu untuk memperkaya tampilan presentasi dan membuatnya lebih menarik bagi audiens.

34. Apa fungsi Chart Styles pada menu Design?​


Jawaban:

PENGERTIAN CHART/GRAFIK

Grafik merupakan suatu rangka yang digunakan untuk membentuk objek visualisasi dari data sebuah table. Grafik disajikan dalam bentuk kombinasi data-data baik berupa angka, huruf, simbol, gambar, lambang, perkataan, lukisan, yang disajikan dalam sebuah media dengan tujuan memberikan gambaran tentang suatu data dari penyaji materi kepada para penerima materi dalam proses menyampaikan informasi.

Grafik/ chart berfungsi unutk menggambarkan data-data kedalam bentuk angka dengan teliti dan dapat mengerangkan perkembangannya dan juga dapat membandingkannya dengan suatu objek atau suatu peristiwa yang saling berhubungan dengan singkat dan jelas.

MACAM-MACAM CHART/GRAFIK

Chart berfungsi untuk menyisipkan atau menampilkan chart wizard untuk pembuatan Graphic di document anda. Ada berbagai macam chart yaitu Column, Line, Pie, Bar, Area, X Y (Scatter), Stock, Surface, Doughnut, Bubble dan juga Radar.
