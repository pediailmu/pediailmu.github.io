[1]
aN₂ + bH₂ → cNH₃

Koefisien kiri = kanan

H ⇒ 2b = 3c
Misal a = 1, maka c = 2
2b = 3(2) ⇒ b = 3
a : b : c = 1 : 3 : 2 
N₂ + 3H₂ → 2NH₃

[2]
aAl + bO₂ → cAl₂O₃

Al ⇒ a = 2c
O ⇒ 2b = 3c
Misal a = 2, c = 1
2b = 3(1) ⇒ b = ³/₂
a : b : c = 2 : ³/₂ : 1 dikalikan 2 ⇒ a : b : c = 4 : 3 : 2
4Al + 3O₂ → 2Al₂O₃

[3]
aNH₃ + bO₂ → cNOH₂O

N ⇒ a = c 
H ⇒ 3a = 2c
O ⇒ 2b = 2c atau b = c
Misal a = 1, c = 1
Tidak berhubungan dengan 3a = 2c
Tak dapat diteruskan, mungkin ada kekurangan di soal

[4]
aMg + bHCl → cMgCl₂ + dH₂

Mg ⇒ a = c
H ⇒ b = 2d
Cl ⇒ b = 2c
Misal a = 1, c = 1
b = 2(1) maka b = 2
2 = 2d maka d = 1
a : b : c : d = 1 : 2 : 1 : 1
Mg + 2HCl → MgCl₂ + H₂

12. Dibantu ya kak,, nanti malam kumpul...Soal kimia kelas 10​


Jumlah mol yang dapat dalam 35,5 gram gas klorin adalah 0,5 mol.

»PEMBAHASAN«

[tex] \boxed{KONSEP \ MOL} [/tex]

mol adalah jumlah partikel yang terkandung dalam suatu zat yang jumlahnya sama dengan banyaknya atom yang terdapat di dalam 12 gram atom C-12.

RUMUS MENCARI MOL:

mol = [tex] \frac{massa}{Mr} [/tex]

Keterangan:

massa = gram

Mr = massa molekul relatif (gram/mol)

=================================

Diketahui:

Massa = 35,5 gram

Mr = 71 gram/mol

Ditanya:

Jumlah mol?

Penyelesaian:

• Gunakan rumus

mol = [tex] \frac{massa}{Mr} [/tex]

• Lalu masukkan data yang sudah diketahui:

mol = [tex] \frac{massa}{Mr} [/tex]

mol = [tex] \frac{35,5 \ gram}{71 \ gram/mol} [/tex]

mol = 0,5 mol

Jadi jumlah mol yang dapat dalam 35,5 gram gas klorin adalah 0,5 mol.

33. Jika Ar Mg = 24, Ar H = 1 dan Ar O = 16. Ar S = 32. Berapakah Mr dari senyawa dengan rumus kimia MgSO4. 5H2O *


Jika Ar Mg = 24, Ar H = 1 dan Ar O = 16. Ar S = 32. Berapakah Mr dari senyawa dengan rumus kimia MgSO₄. 5H₂O adalah 142 gram/mol

Penjelasan:

Jawab:

Mr MgSO₄ = Ar Mg + Ar S + 4(Ar O)

Mr MgSO₄ = 24 + 32 + 4(16)

Mr MgSO₄ = 120 gram/mol

Mr 5H₂O = 5(Ar H) + Ar O

Mr 5H₂O = 5 (1) + 16

Mr 5H₂O = 6 + 16

Mr 5H₂O = 22 gram/mol

Maka, Mr MgSO₄. 5H₂O = 120 gram/mol + 22 gram/ mol = 142 gram/mol

Pembahasan:  

Molekul adalah pergabungan dari beberapa atom unsur dengan suatu perbandingan tertentu. Unsur-unsur yang sama bergabung membentuk molekul suatu unsur, sedangkan suatu unsur-unsur yang berbeda membentuk molekul-molekul senyawa. Massa molekul relatif suatu unsur ataupun senyawa dinyatakan oleh massa molekul relatif (Mr). Massa molekul relatif merupakan suatu perbandingan massa molekul unsur atau senyawa terhadap 1/12 × massa atom C-12. Rumus penentuan massa molekul relatitif sebagai berikut.  

Mr (Unsur) = [tex]\frac{Massa~Molekul~unsur}{1/12~x~massa~atom~C-12}[/tex]

Mr (Senyawa) = [tex]\frac{Massa~Molekul~Senyawa}{1/12~x~massa~atom~C-12}[/tex]

Berdasarkan pengertian molekul yang dinyatakan bahwa molekul ialah gabungan dari atom-atomnya, maka Mr merupakan jumlah Ar atom-atom unsur penyusunnya. Rumus massa molekul relatif adalah sebagai berikut.  

Mr = Σ Ar

PENDAHULUAN

Orde reaksi adalah besarnya faktor konsentrasi zat reaktan yang mempengaruhi laju reaksi. Orde reaksi dinyatakan dalam bilangan berpangkat dari konsentrasi reaktan dalam persamaan laju reaksinya.

Jika konsentrasi salah satu zat dinaikkan a kali, dan laju reaksi naik b kali, maka:

[tex] \boxed { \rm {{a}^{orde} \: = \: b}}[/tex]

Persamaan laju reaksi hanya bisa dinyatakan setelah melakukan eksperimen, dengan demikian laju reaksi (pada umumnya) tidak dapat dinyatakan hanya dari persamaan reaksi.

Reaksi A + B → AB, persamaan laju reaksinya adalah:

[tex] \boxed{ \rm {v \: = \: k \: {[A] }^{m} \: {[B] }^{n} }}[/tex]

dengan:

v = laju reaksi

k = tetapan laju reaksi

m = orde reaksi terhadap A

n = orde reaksi terhadap B

[tex] \boxed{ \rm{Orde \: total \: = \: m \: + \: n}}[/tex]

DIKETAHUI:

Hasil pengukuran laju reaksi

DITANYA:

a. Orde total

b. Persamaan laju reaksi

••Untuk menentukan orde reaksi terhadap A cari konsentrasi B yang sama, yakni terletak pada data nomor 2 dan 3.

[tex] \rm { \dfrac{t_3}{t_2} \: = \: \dfrac{k \: . \:{[A]_2 }^{m} \: . \:{ [B]_2 }^{n} }{k \: . \:{[A]_3 }^{m} \: . \: {[B]_3 }^{n} } }[/tex]

[tex]\rm{ 1\: = \: \dfrac{ {[0,1] }^{m} }{{[0,2] }^{m} \: } } [/tex]

[tex] \rm {1 \: = \: {(\dfrac{1}{2} ) }^{m}} [/tex]

[tex] \rm{( { \cancel{\dfrac{1}{2} })}^{0} \: = \: {( \cancel{\dfrac{1}{2} })}^{m} }[/tex]

[tex] \rm{0 \: = \: m}[/tex]

[tex] \rm{m \: = \: 0}[/tex]

Menentukan orde reaksi terhadap B

••Untuk menentukan orde reaksi terhadap B cari konsentrasi A yang sama, yakni terletak pada data nomor 1 dan 2.
