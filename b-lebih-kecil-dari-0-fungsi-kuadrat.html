3. jika fungsi kuadrat melalui titik-titik A(0, -6), B(-1, 0) dan C(1, -10). tentukan rumus fungsi kuadratnya


[tex]y=ax^2+bx+c[/tex]
Karena melalui A(0,-6), maka:
[tex]-6=a(0)^2+b(0)+c[/tex]
[tex]c=-6[/tex]
Karena melalui B(-1,0), maka:
[tex]0=a(-1)^2+b(-1)-6[/tex]
[tex]0=a-b-6[/tex]
[tex]a-b=6[/tex]...(1)
Karena melalui C(1,-10), maka:
[tex]-10=a(1)^2+b(1)-6[/tex]
[tex]-10+6=a+b[/tex]
[tex]a+b=-4[/tex]...(2)
Eliminasi variabel b pada persamaan (1) dan (2), seperti berikut:
a - b = 6
a + b = - 4
--------------- (+)
2a = 2
a = 1
a + b = - 4
b = - 4 - a = - 4 - 1 = - 5
Dengan demikian fungsi kuadrat tersebut adalah:
[tex]y=x^2-5x-6[/tex]

21. suatu grafik fungsi kuadrat memotong sumbu x di A(1, 0) dan B(2, 0). Jika grafik fungsi kuadrat itu melalui titik (0, 4) dan puncaknya di titik (p, q) maka p + q = ... - Brainly #Backtoschool2019


Fungsi kuadrat

Memotong sumbu [tex]x[/tex] di [tex]A(1,0)[/tex] dan [tex]B(2,0)[/tex]

Melalui titik [tex](0,4)[/tex]

Titik puncaknya [tex](p,q)[/tex]

[tex]p+q=\dots ?[/tex]

Penyelesaian:

Fungsi kuadrat yang melalui 2 titik dan memotong sumbu [tex]x[/tex] adalah

[tex] \displaystyle y = a(x-x_1)(x-x_2)[/tex]

Diketahui:

[tex] \displaystyle A(1,0) \to x_1 = 1 \\ B(2,0) \t x_2 = 2 [/tex]

Masukkan [tex]x_1[/tex] dan [tex]x_2[/tex] tersebut

[tex] \displaystyle y = a(x-x_1)(x-x_2) \\ y = a(x-1)(x-2) [/tex]

Mencari nilai [tex]a[/tex] dengan memasukkan titik lain yang melalui fungsi kuadrat tersebut,

titik [tex](0,4) \to x=0; \:\: y=4; [/tex]

[tex] \displaystyle 4 = a(0-1)(0-2) \\ a(-1)(-2) = 4 \\ 2a = 4 \\ a = 2 [/tex]

Sehingga fungsi kuadratnya

[tex] \displaystyle y = 2(x-1)(x-2) \\ y = 2(x^2-3x+2) \\ y = 2x^2-6x+4 [/tex]

Dari persamaan fungsi kuadrat tersebut, diperoleh

[tex] a = 2 \\ b = -6 \\ c = 4[/tex]

28. Diberikan fungsi kuadrat f(x) = ax2 + bx + c dengan a, b, dan c merupakan bilangan real. Jika a > 0; b < 0 dan c > 0, maka grafik fungsi kuadrat f(x) yang mungkin adalah ....


Diberikan fungsi kuadrat f(x) = ax² + bx + c dengan a, b, dan c merupakan bilangan real. Jika a > 0; b > 0 dan c > 0, maka grafik fungsi kuadrat f(x) yang mungkin adalah ...

Jawaban Pendahuluan

Soal matematika ini berkaitan dengan materi fungsi kuadrat.

Pembahasan

Persamaan kuadrat mempunyai bentuk umum yaitu ax^2 + bx + c = 0, dimana a adalah koefisien dari variabel x^2, b adalah koefisien dari variabel x^1, dan c adalah koefisien dari variabel x^0.

[tex]ax^2 + bx + c = 0[/tex]

Dalam persamaan kuadrat, rumus umum yang digunakan dalam koefisien persamaan kuadrat adalah nilai diskriminan, yang dinotasikan dengan D dan dirumuskan dengan D = b^2-4ac.

[tex]D = b^2-4ac[/tex]

Jenis bilangan pada akar-akar persamaan kuadrat ditentukan oleh nilai diskriminannya melalui koefisiennya.

Akar riil/nyata, maka D ≥ 0 Akar riil berlainan, maka D > 0 Akar riil sama/kembar, maka D = 0 Akar tidak riil (imajiner), maka D < 0

Akar-akar persamaan kuadrat dapat dihitung dengan rumus umum yang melibatkan koefisiennya sebagai berikut:

[tex]x_{1} = \frac{-b + \sqrt{b^2 - 4ac}}{2a} = \frac{-b + \sqrt{D}}{2a}\\x_{2} = \frac{-b - \sqrt{b^2 - 4ac}}{2a} = \frac{-b - \sqrt{D}}{2a}[/tex]  

[tex]x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}\\x = \frac{-b \pm \sqrt{D}}{2a}[/tex]  

Selain itu juga terdapat rumus pendukung yang melibatkan koefisiennya persamaan kuadrat.

Penjumlahan akar-akar persamaan kuadrat

x1 + x2 = -b/a

[tex]x1 + x2 = -\frac{b}{a}[/tex]

Perkalian akar-akar persamaan kuadrat

x1 ⋅ x2 = c/a

[tex]x1 * x2 = \frac{c}{a}[/tex]

Selisih akar-akar persamaan kuadrat

x1 - x2 = √D/a

[tex]x1 - x2 = \frac{\sqrt{D}}{a} = \frac{\sqrt{b^2 - 4ac}}{a}[/tex]

Fungsi kuadrat mempunyai bentuk umum yaitu f(x) = ax^2 + bx + c, dimana a adalah koefisien dari variabel x^2, b adalah koefisien dari variabel x^1, dan c adalah koefisien dari variabel x^0.

Secara grafis, fungsi kuadrat digambarkan dalam kurva parabola atau cekungan panjang pada koordinat kartesius yang berada di antara sumbu x (garis mendatar) dan sumbu y (garis meninggi).

Umumnya, jika titik potong kurva berada di sumbu x, maka nilai y adalah nol (y=0), sedangkan jika titik potong kurva berada di sumbu y, maka nilai x adalah nol (x=0).

Kurva parabola untuk fungsi kuadrat f(x) = ax^2 + bx + c  memiliki sumbu simetri dan titik puncak pada titik koordinat tertentu.

Sumbu simetri (xp)

[tex]x_{p} = \frac{-b}{2a}[/tex]

Titik puncak (yp)

[tex]y_{p} = \frac{-D}{4a} = -\frac{b^2-4ac}{4a}[/tex]

Ciri-ciri parabola untuk fungsi kuadrat adalah sebagai berikut.

Bentuk parabola tegak dan simetris. Bentuk cekungan parabola terbuka ke atas untuk a > 0 dan terbuka ke bawah untuk a < 0. Sumbu simetri berada pada titik terhadap sumbu x (xp) dan titik puncak berada pada titik terhadap sumbu y (yp).

Bentuk grafik fungsi kuadrat ditentukan oleh nilai diskriminannya melalui koefisiennya yaitu D = b^2-4ac.

Grafik memotong sumbu x, maka D > 0 Grafik menyinggung sumbu x, maka D = 0 Grafik tidak memotong sumbu x, maka D < 0

Rumus-rumus fungsi kuadrat adalah sebagai berikut.

Untuk titik potong sumbu x dan satu titik sembarang (x, y)

[tex]y = a(x - x_1)(x - x_2)[/tex]

Untuk titik puncak dan satu titik sembarang (x, y)

[tex]y = a(x - x_p)+y_p^2[/tex]

Diketahui:

f(x) = ax² + bx + c

a > 0, b > 0, c > 0

a,b,c ∈ R

Ditanya:

Grafik fungsi kuadrat yang mungkin?

Jawab:

Nilai koefisien fungsi kuadrat yaitu a > 0, sehingga bentuk cekungan parabola terbuka ke atas dan dengan nilai diskriminan atau D = b^2-4ac, maka grafik tidak memotong sumbu x untuk D < 0, menyinggung sumbu x untuk D = 0, atau memotong sumbu x untuk D > 0.

Kesimpulan

Jika fungsi kuadrat f(x) = ax² + bx + c dengan a, b, dan c merupakan bilangan real, serta a > 0; b > 0 dan c > 0, maka grafik fungsi kuadrat f(x) yang mungkin adalah bentuk cekungan parabola terbuka ke atas dan mempunyai tiga kemungkinan grafik terhadap sumbu x.

Fungsi kuadrat mempunyai bentuk umum yaitu f(x) = ax^2 + bx + c, dimana a adalah koefisien dari variabel x^2, b adalah koefisien dari variabel x^1, dan c adalah koefisien dari variabel x^0.

Secara grafis, fungsi kuadrat digambarkan dalam kurva parabola atau cekungan panjang pada koordinat kartesius yang berada di antara sumbu x (garis mendatar) dan sumbu y (garis meninggi).

Umumnya, jika titik potong kurva berada di sumbu x, maka nilai y adalah nol (y=0), sedangkan jika titik potong kurva berada di sumbu y, maka nilai x adalah nol (x=0).

Kurva parabola untuk fungsi kuadrat f(x) = ax^2 + bx + c  memiliki sumbu simetri dan titik puncak pada titik koordinat tertentu.

Sumbu simetri (xp)

[tex]x_{p} = \frac{-b}{2a}[/tex]

Titik puncak (yp)

[tex]y_{p} = \frac{-D}{4a} = -\frac{b^2-4ac}{4a}[/tex]

Ciri-ciri parabola untuk fungsi kuadrat adalah sebagai berikut.

Bentuk parabola tegak dan simetris. Bentuk cekungan parabola terbuka ke atas untuk a > 0 dan terbuka ke bawah untuk a < 0. Sumbu simetri berada pada titik terhadap sumbu x (xp) dan titik puncak berada pada titik terhadap sumbu y (yp).

Bentuk grafik fungsi kuadrat ditentukan oleh nilai diskriminannya melalui koefisiennya yaitu D = b^2-4ac.

Grafik memotong sumbu x, maka D > 0 Grafik menyinggung sumbu x, maka D = 0 Grafik tidak memotong sumbu x, maka D < 0

Rumus-rumus fungsi kuadrat adalah sebagai berikut.

Untuk titik potong sumbu x dan satu titik sembarang (x, y)

[tex]y = a(x - x_1)(x - x_2)[/tex]

Untuk titik puncak dan satu titik sembarang (x, y)

[tex]y = a(x - x_p)+y_p^2[/tex]

Diketahui:

f(x) = ax² + bx + c

a > 0, b > 0, c > 0

a,b,c ∈ R

Ditanya:

Grafik fungsi kuadrat yang mungkin?

Jawab:

Nilai koefisien fungsi kuadrat yaitu a > 0, sehingga bentuk cekungan parabola terbuka ke atas dan dengan nilai diskriminan atau D = b^2-4ac, maka grafik tidak memotong sumbu x untuk D < 0, menyinggung sumbu x untuk D = 0, atau memotong sumbu x untuk D > 0.

Kesimpulan

Jika fungsi kuadrat f(x) = ax² + bx + c dengan a, b, dan c merupakan bilangan real, serta a > 0; b > 0 dan c > 0, maka grafik fungsi kuadrat f(x) yang mungkin adalah bentuk cekungan parabola terbuka ke atas dan mempunyai tiga kemungkinan grafik terhadap sumbu x.
