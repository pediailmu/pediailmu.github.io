3. 1. Jelaskan apa tugas profesi dalam pengembangan aplikasi web berikut : a. Web Designer b. Web Programmer c. Web Administrator d. Web Master


Kelas: IX

Mata Pelajaran: Teknologi Informasi

Materi:  World Wide Web        

Kata Kunci:  web, http, internet

 

Tugas profesi dalam pengembangan aplikasi web berikut:


a. Web Designer: merancang susunan dan tampilan suatu situs pemrograman web


b. Web Programmer: membuat sistem dan halaman web sesuai dengan rancangan, dengan menggunakan bahasa pemrograman web


c. Web Administrator: menjaga kelangsungan kerja sistem web sehari-hari


d. Web Master: tugasnya adalah gabungan dari ketiga tugas di atas

 

Jawaban panjang:

 

a. Web Designer:

 

Ketika suatu web akan dibuat, maka disusun terlebih dahulu rancangan bentuk halaman, serta susunan alur halaman dalam bentuk link dari satu halaman ke halaman lain. Jika web menggunakan sistem halaman dinamik, maka database yang digunakan juga perlu dirancang

Merancang susunan dan tampilan suatu situs pemrograman web ini adalah tugas dari web designer.


b. Web Programmer:

 

Ketika sudah ada rancangan bentuk halaman dan alur situs, maka yang membuat sistem dan halaman web sesuai dengan rancangan, dengan menggunakan bahasa pemrograman web adalah Web Programmer. Bahasa pemrograman yang digunakan adalah HTML dengan dapat ditambah dengan Ruby, PHP atau ASP untuk logika halaman, CSS untuk menata tampilan halaman web, SQL untuk menghubungkan halaman web dengan database serta JavaScript untuk tampilan interaktif.


c. Web Administrator:

 

Ketika halaman web sudah jadi maka yang menjaga kelangsungan kerja sistem web sehari-hari adalah Web Administrator. Halaman web memerlukan orang yang rutin mengecek jalanya web apakah ada permasalahan atau tidak.


d. Web Master:

 

Web master tugasnya adalah gabungan dari ketiga tugas di atas. Web master harus memiliki skill yang dibutuhkan untuk melakukan desain, pembuatan dan perawatan situs web.

12. Di bntu ya kak Apa itu pengembangan aplikasi berbasis web? Jelaskan arsitektur dasar aplikasi web dan sebutkan beberapa teknologi yang digunakan dalam pengembangan aplikasi berbasis web.


Jawaban:

Pengembangan aplikasi berbasis web adalah proses menciptakan dan mengembangkan aplikasi yang dapat diakses melalui web browser menggunakan jaringan internet. Aplikasi web ini berjalan di server dan dapat diakses oleh pengguna melalui peramban web mereka tanpa perlu menginstal aplikasi secara lokal di perangkat mereka.

Arsitektur dasar aplikasi web terdiri dari tiga komponen utama:

1. Front-End (Klien): Komponen front-end bertanggung jawab untuk interaksi langsung dengan pengguna. Ini adalah bagian yang tampak oleh pengguna melalui tampilan antarmuka (user interface/UI). Front-end terdiri dari bahasa markah (HTML), gaya (CSS), dan skrip (JavaScript). HTML digunakan untuk membuat struktur dan elemen halaman web. CSS digunakan untuk mengatur tampilan dan desain halaman web. JavaScript memberikan interaktivitas dan fungsionalitas pada aplikasi web.

2. Back-End (Server): Komponen back-end berada di sisi server dan bertanggung jawab untuk memproses permintaan dari klien, berinteraksi dengan database, dan melakukan logika bisnis yang diperlukan. Server back-end menghasilkan dan mengirimkan data yang diminta oleh klien. Bahasa pemrograman seperti JavaScript (Node.js), PHP, Python, Ruby, dan Java umum digunakan dalam pengembangan back-end.

3. Database: Database digunakan untuk menyimpan dan mengelola data yang diperlukan oleh aplikasi web. Data ini dapat berupa informasi pengguna, konten aplikasi, atau informasi lain yang relevan. Teknologi basis data seperti MySQL, Oracle, MongoDB, dan PostgreSQL digunakan untuk menyimpan dan mengatur data dalam aplikasi web.

Beberapa teknologi lain yang digunakan dalam pengembangan aplikasi web meliputi:

- Framework: Framework seperti React, Angular, dan Vue.js digunakan untuk mempercepat pengembangan aplikasi web dengan menyediakan alat dan struktur yang telah siap untuk digunakan.

- RESTful API: Representational State Transfer (REST) adalah pendekatan untuk membangun API (Application Programming Interface) yang dapat digunakan oleh aplikasi web untuk berkomunikasi dengan server dan mengambil atau memperbarui data.

- Pyton diketahui sebagai bahasa pemrograman yang serba bisa baik itu untuk pembuatan aplikasi desktop, mobile dan web.

- Cloud Computing: Layanan cloud seperti Amazon Web Services (AWS) dan Google Cloud Platform (GCP) digunakan untuk menyimpan dan mengelola aplikasi web di lingkungan cloud, yang menyediakan skalabilitas, ketersediaan, dan keamanan yang diperlukan.

- Masyarakat pembangunan API menggunakan spesifikasi OpenAPI dan pendekatan pembangunan API terbaik.

- Beberapa alat pengembangan web yang populer termasuk Git untuk kerjasama pengembangan kode, Docker untuk pengelolaan kontainer, dan Jenkins untuk otomatisasi pengembangan dan penyiapan.


13. Profesi dalam pengembangan web, kecuali


Profesi dalam pengembangan web kecuali web browser. Menurut versi lengkap dari pertanyaan ini, pilihan jawaban yang disediakan adalah web developer, web programmer, web designer, web administrator dan web browser. Oleh karena itu, web browser bukan profesi dalam pengembangan web, melainkan merupakan perangkat lunak yang membantu pengguna untuk melakukan penjelajahan internet  seperti contohnya: Mozilla Firefox, Chrome, Opera, dan Internet Explorer. Sedangkan, web developer, programmer, designer dan administrator adalah profesi karena dilakukan langsung oleh pengguna sesuai peran masing-masing.

Pembahasan

Dalam pengembangan suatu web, diperlukan keterlibatan pengguna /orang menurut perannya masing-masing sebagai pihak yang terlibat dalam proses pengembangannya. Profesi dalam pengembangan web ini saling memiliki keterkaitan antar fungsinya.  Berikut penjelasan profesi dalam pengembangan web:

Web Developer

Web developer adalah orang yang membuat website dari nol dan umumnya web developer akan menguasai skill dalam web design sebagai front-end dan juga web-programming pada sisi back-end.

Web programmer

Web programmer adalah orang yang terlibat dalam melakukan pemrograman untuk dalam pengembangan suatu website, dan sering kali, seorang web programmer identic dengan orang yang berada pada back-end.

Web designer

Web designer adalah orang  yang mendesain dan merancang suatu website terutama pada sisi front-end atau tampilan website. Seorang web designer tentu harus memiliki kreatifitas untuk mendesain website yang menarik dan ramah bagi pengguna

Web Administrator

Web Administrator adalah orang yang berperan dalam menjaga operasional suatu website baik operasional pada front-end maupun back-end. Seorang web administrator juga perlu mengevaluasi jalannya website sehingga dapat diiketahui masalah yang mungkin terjadi sehingga dapat segera ditangani.

20. alur pengembangan aplikasi web itu apa apa aja :(


Fase requirement, sama dengan metodologi diatas. Pada fase ini saya mencoba mencari tahu beberapa pertanyaan berikut:apa yang dibutuhkan?apa tujuan dari aplikasi ini?apa yang ingin dicapai?apakah ada referensi atau contoh?siapa sasaran penggunaan aplikasi ini?Setelah memperoleh jawaban dari pertanyaan itu maka saya baru memulai fase berikutnya.Fase penentuan feature dan fungsi, pada tahapan ini berdasarkan hasil penelusuran kebutuhan dari aplikasi maka saya menentukan feature dan fungsi apa saja yang akan dibuat untuk memenuhi kebutuhan yang ada. Feature dibagi menjadi feature utama atau bagian besar terlebih dahulu baru kemudian didefenisikan menjadi bagian yang lebih kecil. Misalnya pada aplikasi CMS salah satu feature utama adalah blog, kemudian feature blog didefenisikan seperti feature tagging, category, comment, dsb. Pada tahapan ini yang perlu diperhatikan adalah apakah daftar feature dan fungsi yang dibuat telah memenuhi kebutuhan yang diinginkan.Fase pengumpulan data, konten, gambar, dan file lain yang dibutuhkan dalam pengembangan aplikasi. Berdasarkan daftar feature dan fungsi yang akan dikembangkan maka saya akan mengumpulkan data-data dan file yang diperlukan untuk feature dan fungsi tersebut.Fase analisa dan perancangan, pada tahapan ini saya mencoba menganalisa data yang telah dikumpulkan pada tahapan sebelumnya dan menuangkan hasil analisa kedalam sebuah rancangan. Misalnya berdasarkan feature yang ada maka saya merancang tabel database yang dibutuhkan oleh tiap feature. Berdasarkan konten dan data  yang ada saya juga melakukan perancangan tampilan aplikasi. Selain itu saya juga menyusun struktur aplikasi seperti menu, sitemap, dsb.Fase coding, pada tahapan ini saya memulai menulis kode program. Biasanya dimulai dengan mengubah template dari file image Firework ke html. Setelah itu membuat kerangka aplikasi seperti struktur folder dan file dan memulai pengembangan fungsi untuk tiap feature.Fase setup and testing, pada tahapan ini saya melakukan setup di server dimana aplikasi web nanti akan akan dijalankan. Tentu setelah melakukan testing di komputer lokal terlebih dahulu dan memastikan bahwa semua fungsi berjalan baik. Pada server juga akan dilakukan testing untuk memastikan aplikasi dapat berjalan baik pada server tersebut. Pada tahapan ini juga akan dilakukan beta test dimana beberapa orang akan mencoba menggunakan aplikasi dan memberikan komentar atau feedback terkait dengan fungsi aplikasi.Fase launching, pada fase ini setelah dipastikan aplikasi berjalan baik pada server maka saya akan melakukan sosialisasi baik di dunia nyata atau maya. Beberapa diantaranya adalah melakukan posting di forum, aplikasi social network seperti facebook, membuat posting di blog, dan sosialisasi lainnya.

Pada dasarnya semua metodologi ini adalah opsional. Jadi bisa digantikan sesuai yang anda mau...

Jawaban:

Sebuah tool untuk mengindeks dan mengunduh konten dari internet, lalu disimpan ke dalam database mesin pencari.

Penjelasan:

Pengertian web crawler–atau sering juga disebut spiders— adalah sebuah tool untuk mengindeks dan mengunduh konten dari internet, lalu disimpan ke dalam database mesin pencari.
