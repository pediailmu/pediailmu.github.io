12. Interval nilai p yang menyebabkan fungsi kuadrat f(x) = (p-2)x^2 + 2px + p + 3 definit positif adalah ... (Ujian Nasional 2013)


Ingat! Syarat fungsi kuadrat definit positif (a > 0, D < 0), maka:

D < 0
b² - 4ac < 0
(2p)² - 4(p - 2)(p + 3) < 0
4p² - 4p² - 4p + 24 < 0
-4p + 24 < 0
-4p < -24
p > 6

Jadi, interval nilai p adalah p > 6.

13. Agar fungsi f(x) = mx^2 + 2mx + (m+2) definit positif, maka nilai m yang memenuhi adalah ... (Ujian Nasional 2013)

D < 0
b² - 4ac < 0
(2m)² - 4(m)(m + 2) < 0
4m² - 4m² - 8m < 0
-8m < 0
m > 0

Jadi, nilai m yang memenuhi adalah m > 0.

14. Interval nilai p yang menyebabkan fungsi kuadrat f(x) = (p-2)x^2 + 2px + p + 3 definit positif adalah ... (Ujian Nasional UN 2013)


f(x) = (p - 2)x² + 2px + (p + 3) 

a = (p - 2)
b = 2p
c = (p + 3)

Syaart definit positif :
a > 0
D < 0

Syarat pertama :
a > 0
p - 2 > 0
p > 2. . . . . . . (1)

Syarat kedua :
D < 0
b² - 4ac < 0
(2p)² - 4 • (p - 2) • (p + 3) < 0
4p² - 4 • (p² + 3p - 2p - 6) < 0
4p² - 4 (p² + p - 6) < 0
4p² - 4p² - 4p + 24 < 0
-4p + 24 < 0
24 < 4p
p < 6. . . . . . . . . (2)

Gabungkan syarat pertama dan kedua, sehingga diperoleh interval 2 < p < 6

Jadi, interval nilai p adalah  2 < p < 6

17. Jika f^-1(x)adalah invers dari fungsi f(x) = (2x-4)/ (x-3), x tidak sama dengan 3, maka nilai f^-1(4) = ... (Ujian Nasional UN 2010)


Ingat kembali:
f(x) = (ax + b) / (cx + d)
f⁻¹(x) = (-dx + b) / (cx - a)

f(x) = (2x - 4) / (x - 3)
f⁻¹(x) = (3x - 4) / (x - 2)
f⁻¹(4) = (3(4) - 4) / (4 - 2) = 4
