11. Tuliskan perintah yang ada pada tab insert dan page layout yang ada pada Microsot Word beserta dengan fungsinya!


Jawaban:

Tab Insert

Cover Page berfungsi untuk menyisipkan format halaman depan.  

Blank Page berfungsi untuk menyisipkan halaman kosong di tempat kursor berada.

Table berfungsi untuk menyisipkan tabel ke dalam dokumen.

Picture berfungsi untuk menyisipkan gambar dari dokumen lain.

Clip Art berfungsi menyisipkan gambar bawaan dari Word 2007

Shapes berfungsi untuk menyisipkan shape yang telah disediakan oleh Word 2007.

Smart Art berfungsi untuk menyisipkan smart Art untuk mengkomunikasikan pesan melalui diagram dan tabel.

Chart berfungsi untuk menyisipkan grafik ke dalam dokumen.

Header berfungsi untuk mengedit tampilan Header yang terletak di atas halaman dokumen.

Footer berfungsi untuk mengedit tampilan Footer yang terletak di bawah halaman dokume

Page Number berfungsi untuk menyisipkan penomoran halaman secara otomatis ke dalam dokumen.

Text Box berfungsi menyisipkan kotak yang bisa diberi teks.

Quick Parts berfungsi untuk menyisipkan aksesoris-aksesoris seperti nomor halaman, Header, atau Footer ke dalam halaman.

Word Art berfungsi untuk menyisipkan kaligrafi ke dalam dokumen.

Drop Cap berfungsi untuk memberikan efek Drop Cap

Date & Time berfungsi untuk menyisipkan tanggal dan waktu.

Object berfungsi untuk menyisipkan gambar yang berasal dari program lain ataupun dari file.

Equation berfungsi untuk menyisipkan simbol atau rumus matematika ke dalam dokumen.

Symbol berfungsi untuk menyisipkan simbol-simbol ke dalam dokumen, baik berupa simbol matematika atau huruf arab

Tab Page Layout

a. Themes berfungsi untuk mengatur tampilan keseluruhan halaman, mulai dari warna, efek,

    dan huruf.

b. Theme Color berfungsi untuk mengubah warna dari Theme yang sedang digunakan.

c. Theme Font berfungsi untuk mengubah jenis huruf dari Theme yang sedang digunakan.

d. Margins berfungsi untuk mengubah ukuran pencetakan, seperti batas pengetikan dan orientasinya.

e. Orientation berfungsi untuk mengubah orientasi pengetikan, apakah mau Landscap atau Potrait.

f. Size berfungsi untuk mengubah ukuran kertas yang sedang kita gunakan.

g. Columns berfungsi untuk mengubah halaman menjadi beberapa kolom.

h. Watermark berfungsi untuk menyisipkan tulisan latar disetiap halaman dokumen yang kita buat.

i. Page Color berfungsi untuk memberi warna latar di dokumen yang kita buat.

j. Page Borders berfungsi untuk memberi bingkai atau garis pada halaman dokumen yang kita buat.

k. Indent Left berfungsi untuk menggeser Indentasi ke kiri.

m. Indent Right berfungsi untuk menggeser Indentasi ke kanan.

n. Spacing Before berfungsi untuk mengatur jarak awal spasi antar paragraf.

o. Spacing After berfungsi untuk mengatur jarak akhir spasi antar paragraf.

p. Position berfungsi mengatur posisi gambar di dalam dokumen.

q. Bring to Front berfungsi untuk mengatur posisi gambar agar berada di depan yang lainnya.

r. Send to Back berfungsi untuk mengatur posisi gambar agar berada di belakang yang lainnya

s. Text Wrapping berfungsi untuk mengubah posisi gambar agar teks bisa mengelilinginya.

t. Align berfungsi mengubah posisi objek yang ada di dokumen.

u. Group berfungsi untuk menyatukan dua gambar menjadi satu gambar.

v. Rotate berfungsi untuk memutar objek yang ada di dalamWord 2007.

Penjelasan:

Tab Insert

Tab Insert berfungsi untuk menyisipkan teks, tanggal atau waktu, simbol, diagram, objek, dan sebagainya. Tab ini berfungsi juga untuk memperindah tampilan dokumen yang kita buat.

Tab Page Layout

Tab Page Layout berfungsi untuk mengatur konfigurasi halaman, mulai dari margin, ukuran kertas, sampai kepada penentuan spasi.

2. Cara Membuat Watermark dari Teks

Catatan: Watermark hanya bisa dilihat di tampilan Print Layout dan Full Screen Reading.

Pada Page Layout tab, Page Background group, klikWatermark.
Lakukan hal berikut:Jika ingin menggunakan watermark yang sudah tersedia di galeri, seperti teksConfidential atau Do Not Copy, klik pada teks tersebut.Jika ingin membuat teks tersendiri, klik Custom Watermark untuk membuka kotak dialog Printed Watermark.

Klik Text watermark dan lakukan hal berikut:Klik pada kotak di sampingText, ketik teks yang diinginkan.Atur format pada Font (tipe font), Size (ukuran), danColor (warna).Atur Layout watermark.Klik OK bila sudah selesai.

34. tombol perintah yang ada pada tab page layout


Tab page layout terdiri dari 5 group yaitu :


1.Themes terdiri dari ikon-ikon Themes, Themes Color, Themes Fonts, dan      Theme Effects.

2.Page Setup tediri dari ikon-ikon Margins, Orientations, Size, Columns,            Break, Line Numbers, dan  Hypenation.

3.Page Background terdiri dari ikon-ikon Watermark, Page Color, dan Page     Borders.

4.Paragraph terdiri dari ikon-ikonn Indent ( Left Right) serta Spacing(Before     dan After)

5.Arrange terdiri dari ikon-ikon Position, Bring to Front, Send to Back, Text        Wrapping, Align, Group, dan Rotate.
