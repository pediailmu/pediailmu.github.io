1. Alat/perangkat jaringan adalah suatu peralatan yang digunakan untuk membentuk suatu jaringan komputer. Alat/perangkat jaringan tersebut memiliki berbagai fungsi, antara lain sebagai berikut:

Router: membantu mengarahkan paket data dari satu jaringan ke jaringan lainSwitch: membantu mengalirkan paket data dari satu perangkat ke perangkat lain dalam suatu jaringanHub: membantu meneruskan paket data dari satu perangkat ke semua perangkat lain dalam jaringanModem: membantu mengubah sinyal digital menjadi sinyal analog dan sebaliknyaWireless Adaptor: membantu menghubungkan perangkat ke jaringan nirkabelServer: membantu memfasilitasi akses bagi banyak perangkat ke sumber daya jaringan (seperti file, printer, dll.

2. Router adalah perangkat jaringan yang membantu mengarahkan paket data dari satu jaringan ke jaringan lain. Router memiliki fungsi sebagai berikut:

Menerima paket data dari satu jaringan dan mengarahkannya ke jaringan lainMenerima paket data dari Internet dan mengarahkannya ke jaringan lokalMembantu membatasi akses ke jaringan dan memfasilitasi akses Internet

3. Fungsi dan cara kerja wireless card adalah sebagai berikut:

Menerima sinyal nirkabel dari router atau access pointMengirim dan menerima paket data melalui jaringan nirkabelMemfasilitasi akses laptop ke sumber daya jaringan (seperti file, printer, dll.) melalui jaringan nirkabel

4. Fungsi dan cara kerja USB WIFI Adaptor adalah sebagai berikut:

Menerima sinyal nirkabel dari router atau access pointMengirim dan menerima paket data melalui jaringan nirkabelMemfasilitasi akses perangkat ke sumber daya jaringan (seperti file, printer, dll.) melalui jaringan nirkabelJenis-jenis wireless adaptor meliputi USB WIFI Adaptor, PCI WIFI Adaptor, dan PCMCIA WIFI Adaptor.

5. Server adalah perangkat jaringan yang berfungsi sebagai pusat layanan bagi sistem jaringan. Server memiliki kemampuan untuk memproses permintaan dari klien (client) dan menyediakan data atau layanan yang dibutuhkan. Server dapat berupa komputer yang khusus ditujukan untuk menjalankan layanan jaringan, seperti web server, file server, atau print server. Fungsi utama dari server adalah untuk memfasilitasi akses dan pertukaran data antara sistem jaringan, memastikan ketersediaan layanan jaringan, dan memastikan konsistensi data yang disimpan.

6. Modem adalah singkatan dari modulator demodulator yang bertugas untuk mentransmisikan data digital menjadi sinyal analog yang bisa diterima oleh jaringan telepon, dan kembali mengubah sinyal analog menjadi data digital. Ada beberapa jenis modem berdasarkan cara pemasangan, yaitu:

Internal modem: merupakan jenis modem yang terpasang secara internal pada motherboard komputerExternal modem: merupakan jenis modem yang berbentuk perangkat luar dan terhubung dengan komputer melalui port USB atau port serial.

7. Bridge adalah perangkat jaringan yang berfungsi sebagai penghubung antar jaringan lokal (LAN) yang berbeda, sehingga memungkinkan komunikasi antar jaringan tersebut.

8.Cara Kerja Switch:

Menerima paket data: Switch menerima paket data dari berbagai perangkat pada jaringan.Melakukan forwarding paket data: Switch memeriksa alamat tujuan dari paket data dan memutuskan bagaimana paket data harus diteruskan.Mengirimkan paket data ke tujuannya: Switch mengirimkan paket data ke perangkat tujuan sesuai dengan alamat tujuan yang terdapat dalam paket data.Menyimpan dan memantau paket data: Switch menyimpan paket data yang tidak diteruskan dan memantau paket data yang masuk dan keluar dari jaringan.

Bedanya dengan Hub:

Alokasi bandwith: Switch memiliki alokasi bandwith yang lebih baik dibandingkan dengan hub, sehingga switch dapat mengelola traffic jaringan secara efisien.Kecepatan: Switch memiliki kecepatan yang lebih cepat dibandingkan dengan hub, karena switch mampu memproses paket data secara parallel.Dapat membedakan alamat tujuan: Switch memiliki kemampuan untuk membedakan alamat tujuan dari paket data, sehingga switch dapat memutuskan bagaimana paket data harus diteruskan. Sedangkan hub hanya meneruskan paket data ke semua perangkat yang terhubung ke hub.Dapat membantu menjaga keamanan jaringan: Switch memiliki kemampuan untuk membatasi akses ke jaringan, sehingga switch dapat membantu menjaga keamanan jaringan.

9. Komputer Client adalah komputer yang bertugas untuk meminta layanan dari server. Fungsi dari komputer client adalah untuk meminta akses data atau layanan dari server, seperti mengakses file, mengirim email, menjalankan aplikasi, dll.

10. Repeater adalah perangkat jaringan yang bertugas untuk memperkuat dan memperpanjang jangkauan sinyal dalam jaringan. Repeater menerima sinyal yang lemah, memperkuat sinyal tersebut, dan mengirimkannya kembali ke jaringan, sehingga memperpanjang jangkauan sinyal.

- Input Device (Keyboard, Mouse, dll): Digunakan untuk memasukkan data dan perintah ke dalam komputer.

- Output Device (Monitor, Printer, Speaker, dll): Digunakan untuk menampilkan hasil pengolahan data atau mengeluarkan informasi dari komputer.

- Alat input: Keyboard, Mouse, Touchscreen

- Alat output: Monitor, Printer, Speaker

- Alat penyimpanan: Hard Disk Drive (HDD), Solid State Drive (SSD), USB Flash Drive
