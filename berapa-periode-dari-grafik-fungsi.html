23. grafik fungsi f(x)= sin 4x mempunyai periode


Grafik fungsi f(x) = sin 4x mempunyai periode sebesar 90ᵒ atau ½ π. Amplitudo fungsi trigonometri merupakan setengah dari selisih nilai terbesar dengan nilai terkecil fungsi tersebut. Bentuk fungsi trigonometri

y = A sin k (x ± b) ± c y = A cos k (x ± b) ± c y = A tan k (x ± b) ± c

dengan

A = amplitudo

Untuk fungsi trigonometri pada sinus dan kosinus

Besar periodenya: p = [tex]\frac{360^{o}}{k}[/tex] atau p = [tex]\frac{2 \pi}{k}[/tex]

Untuk fungsi trigonometri pada tangen

Besar periodenya: p = p = [tex]\frac{180^{o}}{k}[/tex] atau p = [tex]\frac{\pi}{k}[/tex]

Pembahasan

f(x) = sin 4x

A = 1 k = 4

Berdasarkan persamaan fungsi trigonometri tersebut

Besar amplitudonya adalah A = 1

Besar periodenya adalah

p = [tex]\frac{360^{o}}{k} = \frac{360^{o}}{4}[/tex] = 90ᵒ (dalam satuan derajat) p = [tex]\frac{2 \pi}{k} = \frac{2 \pi}{4} = \frac{\pi}{2} = \frac{1}{2} \pi [/tex] (dalam satuan π radian)

27. Tentukan Amplitudo dan Periode grafik fungsi trigonometri y = 3 cos 2 Dan juga membuat grafik fungsi


PEMBAHASAN

Diketahui fungsi seperti berikut ini :

f(x) = 3 cos 2x°

Marilah kita mencoba mencari beberapa nilai - nilai yang penting dari grafik fungsi ini :

Untuk mencari Amplitudo tinggal di lihat koefisien di depan fungsi kosinus nya yakni angka 3 , jadi Amplitudo untuk fungsi ini adalah 3.

f(x) = 3 cos 2x

Untuk mencari Periode tinggal di bagi 360° dengan koefisien variabel x yakni 2 di fungsi ini , jadi Periodenya adalah 360° / 2 = 180°

f(x) = 3 cos 2x

Untuk mencari Nilai maksimum dan minimum tinggal di masukkin nilai maksimum fungsi kosinus yakni 1 dan nilai minimum fungsi kosinus yakni -1 , sehingga persamaan menjadi :

f(x) maksimum → 3 cos 2x = 3 (1) = 3

f(x) minimum → 3 cos 2x = 3 (-1) = -3

Grafik fungsi nya bisa di gambarkan seperti terlihat pada gambar di lampiran.

29. Periode grafik fungsi f(x) = 2 cos (²x + π) adalah​


Jawaban:

π/2

Penjelasan dengan langkah-langkah:

Untuk mencari periode grafik fungsi f(x) = 2 cos (²x + π), maka perlu dicari dulu nilai dari²x + π.

Karena²x + π merupakan suku konstan dari²x, maka²x + π akan memiliki periode yang sama dengan periode dari²x.

Periode dari²x adalah 2π/² = 2π/4 = π/2, maka periode dari²x + π adalah π/2.

Maka periode grafik fungsi f(x) = 2 cos (²x + π) adalah π/2.
