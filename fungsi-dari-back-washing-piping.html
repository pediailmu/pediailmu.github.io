27. Jelaskan fungsi dari tombol back, forward, home


Pada aplikasi web browser (Mozilla Firefox, Chrome, Internet Explorer, dsb),
- Tombol Back berfungsi untuk kembali ke halaman web sebelumnya. Misal kita sedang ada di halaman 2 suatu web, jika ingin kembali ke halaman 1, tinggal klik tombol Back
- Tombol Forward berfungsi untuk maju ke halaman web selanjutnya. Misal jika kita sudah ada di halaman 1 (masih berkaitan dengan contoh yg sebelumnya), dan ingin balik lagi ke halaman 2 tinggal klik tombol Forward 
- Tombol Home berfungsi untuk kembali ke Homepage/halaman awal dari web browser tersebut

39. Software Go Back difungsikan untuk?


Norton GoBack (Dahulu dikenal sebagai WildFile GoBack, Adaptec GoBack, dan Roxio GoBack)
Berbasis Microsoft Windows disk utilitas yang dapat merekam hingga 8Gb perubahan disk. Bila file system idle selama beberapa detik, itu menandai ini sebagai "titik aman". Memungkinkan produk disk drive untuk kembali ke titik manapun dalam riwayat yang tersedia. Hal ini juga memungkinkan versi file yang akan dipulihkan, dan versi sebelumnya dari seluruh disk untuk dapat melihat-lihat. Tergantung pada aktivitas disk, riwayat yang khas mungkin mencakup beberapa jam sampai beberapa hari.
