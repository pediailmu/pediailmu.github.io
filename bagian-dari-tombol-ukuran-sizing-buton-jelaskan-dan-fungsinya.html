8. sebutkan dan jelaskan fungsi tombol tombol tool ukuran jendela dalam microsoft word 2007 !


Fungsi Tombol Minimize 


Anda tentunya sudah melihat bentuk tombol ini pada gambardi atas. Tombol ini adalah berguna untuk memberikan perintah untuk memperkecil ukuran jendela dari Microsoft Word sehingga jendela microsoft word akan mengecil dan menjadi sebuah tombol yang aktif pada taskbar yang ketika tombol ini di klik maka jendela microsoft word akan tampil pada layar monitor anda. 


Fungsi Tombol Restore 


Tombol restore pada jendela microsoft word berfungsi mengembalikan ukuran jendela kembali ke ukuran semula yaitu sebelum tombol restore ini aktif. 


Fungsi Tombol Maximize 


Fungsi tombol ini adalah untuk memksimalkan ukuran jendela dari program aplikasi microsoft word. Fungsi ini adalah merupakan fungsi yang berlawanan dari tombol ukuran jendela Minimize yang mana fungsi tombol ukuran jendela minimize adalah untuk memperkecil ukuran jendela. 


Fungsi tombol Close 


Dalam bahasa inggris arti dari kosa kata close adalah menutup. Jadi tombol close pada tombol ukuran jendela microsoft word mempunyai fungsi menutup jendela Microsoft Word.

30. sebutkan 3 bagian tombol yg ada di mouse dan jelaskan fungsi nya!


3 bagian tombol yg ada di mouse adalah left button (tombol kiri), right button (tombol kanan) dan mouse wheel (roda mouse). Left button berfungsi untuk menunjuk satu menu tertentu dan eksekusi perintah yang ada di dalamnya. Right button berfungsi memunculkan menu tambahan yang ada dalam point tertentu. Sedangkan mouse wheel berfungsi untuk scrolling (memutar/menggulung) halaman.

Pembahasan

Mouse yang sering kita gunakan adalah alat pointer (penunjuk) menu yang berfungsi sebagai interface (perantara) antara manusia sebagai user dan komputer sebagai alatnya. Komputer membutuhkan masukan perintah dari manusia tentang apa saja yang akan dilakukannya, maka dalam hal ini alat yang menjadi perantara adalah mouse dan keyboard. Ketika komputer misalnya memberikan output di layar monitor berupa tombol-tombol apa saja yang harus diklik, maka dia menunggu masukan dari manusia melalui mouse dan keyboard. Ketika kita click (pencet) tombol mouse atau keyboard maka komputer mendapatkan perintah dari kita apa yang harus dia lakukan, misalnya membuka link tertentu atau melaksanakan operasi tertentu. Tanpa mouse dan keyboard kita tidak bisa berinteraksi dengan komputer. Dalam hal ini mouse mempunyai 3 tombol yang masing-masing ada fungsinya. Tombol kiri berfungsi sebagai tombol utama yang akan melaksanakan perintah tertentu ketika kita klik di suatu posisi di layar monitor. Tombol kanan berfungsi memunculkan menu tambahan untuk memodifikasi, sedangkan mouse wheel berfungsi sebagai alat pemutar, untuk memajukan atau memundurkan halaman di dalam layar.

Maaf kallo sallahKeyboard merupakan salah satu jenis input device yang paling tua dan telah lama melengkapi suatu sistem perangkat komputer.

1. Numeric Keypad
Numeric keypad merupakan bagian khusus dari keyboard yang berisi angka dan sangat berfungsi untuk memasukkan data berupa angka dan operasi perhitungan. Struktur angkanya disusun menyerupai kalkulator dan alat hitung lainnya.
2. Tombol kontrol (Control keys)
Tombol ini menyediakan kontrol terhadap kursor dan layar. Tombol yang termasuk dalam kategori ini adalah 4 tombol bersimbol panah di antara tombol ketik dan numeric keypad, home, end, insert, delete, page up, page down, control (ctrl), alternate (alt) dan escape (esc).
3. Backspace adalah tombol untuk menghapus dari kanan ke kiri
4. ESC(Escape) kegunaannya sama dengan cancel yaitu membatalkan suatu tahap pekerjaan
5. Arah berfungsi menggerakkan penunjuk karakter/kursor sesuai arah panah (←↑→↓).
6. Tab berarti pindah field atau daerah isian berikutnya,atau lompat ke daerah penghentian berikutnya.
7. Home akan mengakibatkan kursor berpindah ke awal baris dimana kursor berada.
8. End akan mengakibatkan kursor berpindah ke akhir baris dimana kursor berada.
9. Page Up akan mengakibatkan kursor berpindah ke atas sejauh satu layar dari kursor berada.
10. Page Down akan mengakibatkan kursor berpindah ke bawah sejauh satu layar dari kursor berada.

40. Jelaskan fungsi-fungsi dari bagian-bagian windows dibawah ini? a. Shortcut b. Tombol start c. Tombol pencarian d. Taskbar e. Background deskop​


Jawaban:

Misalkan pada Windows 10:

a. Shortcut berfungsi untuk memasuki program/ aplikasi secara cepat. Pengguna windows dapat membuka program/ aplikasi dg melakukan dobel klik mouse di shortcut yg tampil di desktop windows

b. Tombol Start berfungsi sebagai tombol yg mulai diklik paling awal bila pengguna Windows ingin membuka program tertentu yg tidak tersedia shortcutnya di desktop windows.

c. Tombol pencarian berfungsi untuk menampilkan kotak pencarian data/ file tertentu (baik file dokumen maupun file sistem) yg diinginkan.

d. Taskbar berfungsi sebagai tempat untuk menampilkan program yg sedang aktif sekaligus sebagai tempat bagi program/ aplikasi dalam kondisi minimize.

e. Background desktop berfungsi sebagai gambar latar belakang tampilan utama Windows.

Penjelasan:

Untuk lebih jelasnya, bisa memperhatikan gambar lampiran:

a. Shortcut dapat ditambah oleh pengguna dg cara klik kanan area desktop dan klik New - Shortcut (ditunjukkan tanda panah merah)

b. Tombol Start ditunjukkan tanda panah putih

c. Tombol Pencarian ditunjukkan tanda panah kuning

d. Taskbar adalah bagian yg ditunjukkan tanda panah coklat

e. Background desktop bisa diatur dg cara klik kanan area desktop dan klik Personalize sehingga akan tampil kotak dialog settings (ditunjukkan tanda panah hitam).
